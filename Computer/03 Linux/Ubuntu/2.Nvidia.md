# 01 Ubuntu如何管理显卡
## 1.1 查看电脑有哪些显卡

```bash
lspci -k | grep -A 2 -i "VGA"

# 或

neofetch
```

## 1.2 查看电脑正在使用哪张显卡

* 设置>关于
* 可以看到电脑正在使用的显卡

## 1.3 安装显卡驱动

### 1.3.1 通过包管理器下载

* NVIDIA可以直接在应用商店中下载 `NVIDIA X Server Settings` 作为显卡的配置中心

* 或者

```bash
sudo apt-get nvidia-settings
```

> [关于软件如何下载](1.软件的安装与管理.md)

### 1.3.2 在官网下载驱动

我们在 [官方驱动](https://www.nvidia.com/en-us/drivers/unix/) 中搜索并查找相对应(`x86_64`)的驱动。

> [!attention] 
> 理论上来说官方列出来的驱动都可以使用，但是，由于从官方下载的驱动需要我们自己编译，并且 Ubuntu 系统的 gcc 包是固定版本，不一定能够符合你选择的驱动版本的编译要求。
> 
> 因此，建议选择 [1.3.1](2.Nvidia.md#1.3.1%20通过包管理器下载) 中所推荐的驱动 ~~（直接通过包管理器安装就省事）~~ 

在下载好驱动安装脚本后，我们直接运行脚本即可 : 

```bash
sudo bash ./NVIDIA-Linux-x86_64-*.sh
```

## 1.4 查看、切换显卡

1. 可以在 `NVIDIA X Server Settings` 中 `PRIME Profiles` 中切换显卡

2. 或者

```bash
sudo prime-select query
# 显示正在使用的显卡

sudo /usr/bin/prime-select intel
sudo /usr/bin/prime-select nvidia
# 切换正在使用的显卡
```

3. 安装完官方驱动后可以使用 `nvidia-smi` 查看显卡信息

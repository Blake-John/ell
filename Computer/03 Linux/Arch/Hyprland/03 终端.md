在 Linux 中，终端的使用体验是所有操作中最为重要的我们选择使用 `kitty` + `fish` 来配置我们的命令行工具，并使用 `starship` 来优化我们的使用体验。

`fish` 的配置可以见 [终端配置](../../Basic/终端配置.md) ，里面有着详细的说明，而 `kitty` 及 `starship` 的配置则如下。

# Kitty

`kitty` 是一款可以使用 GPU 渲染并加速的终端模拟器，其功能十分强大，具体信息可见 [kitty](https://sw.kovidgoyal.net/kitty/) 官方文档。

我们对 `kitty` 的配置主要包括背景样式，颜色样式，字体等的配置，我们可以从默认配置出发，并重新设置我们的内容。配置文件存放在 `~/.config/kitty/kitty.conf` ，主要配置如下 : 

```conf
# font
font_family      CaskaydiaCove Nerd Font Mono
bold_font        auto
italic_font      auto
bold_italic_font auto
font_size 11.0
modify_font cell_width 105%
modify_font cell_height 105%

# enhanced usage
copy_on_select clipboard

# appearence
window_padding_width 15
tab_bar_edge top
tab_bar_style powerline
tab_fade 0.25 0.5 0.75 1
tab_powerline_style angled
background_opacity 0.7

# color style
include ./theme-light.conf
```

# Starship

`starship` 可以保证我们在各个不同的 Shell ，不同的终端模拟器上得到一样的外观。

```bash
sudo pacman -S starship
```

# 01 用户及用户组管理

我们可以通过 `cat /etc/group` 来查看当前具有的用户组。如果要新建用户组，我们可以通过 `groupadd` 命令来实现 : 

```bash
sudo group add <group_name>
sudo group add wheel
```

我们可以创建一个用户并加入到目标用户组中 : 

```bash
useradd -m -G <grou_name> <username>
useradd -m -G wheel blake
```

其中 `-m` 选项创建了用户的家目录。

或者，我们可以将用户加入到某一用户组中，通过改变用户的 mod 来实现 : 

```bash
sudo usermod blake -
```

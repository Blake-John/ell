# 02 如何学

学习 Docker 主要有以下几个途径 : 

- B 站视频学习
- Docker 官方文档
- 参考书阅读

## 2.1 B 站视频教程

在B站，首先推荐观看 [30 Min Into Docker](https://www.bilibili.com/video/BV14s4y1i7Vf/?spm_id_from=333.337.search-card.all.click&vd_source=0f7bf6e98597c64847b8d689f7a280c6) 以及 [BlakeHorse - Docker](https://www.bilibili.com/video/BV1HP4118797?spm_id_from=333.788.videopod.episodes&vd_source=0f7bf6e98597c64847b8d689f7a280c6&p=14) 。前面的教程是为了让自己有一个对 Docker 大概的了解，而后一个教程则是一个比较完善的系统教学，能够满足我们的学习目标。

- [30 Min Into Docker](https://www.bilibili.com/video/BV14s4y1i7Vf/?spm_id_from=333.337.search-card.all.click&vd_source=0f7bf6e98597c64847b8d689f7a280c6) 每一集的时间较短，适合在碎片化或者电脑不在手上的时候观看，主要用于了解 ？Docker 以及对 Docker 最简单的使用有一个认知。
- [BlakeHorse - Docker](https://www.bilibili.com/video/BV1HP4118797?spm_id_from=333.788.videopod.episodes&vd_source=0f7bf6e98597c64847b8d689f7a280c6&p=14) 则是一个相对完整的教学视频，时间较长，包含了基础教学以及项目部署，因此需要留出足够的时间来学习，记录笔记，并进行项目实战。

## 2.2 官方文档

官方文档 [Docker Docs](https://docs.docker.com/) 也提供了一个十分全面且完善的帮助文档，分层次，分章节地提供了帮助手册，推荐在其中阅读 [Get Start](https://docs.docker.com/get-started/) 章节，并跟着进行实践，然后，再去仔细看看各个 Concepts 以及 Compose 的用法

## 2.3 参考书

[Learn Docker in a Month of Lunches (Elton Stoneman) (Z-Library)](../../Computer/15%20Docker/Learn%20Docker%20in%20a%20Month%20of%20Lunches%20(Elton%20Stoneman)%20(Z-Library).pdf) 是一本十分全面的参考书，对于我们的目标而言，我们只需要阅读其中的 PART1 以及 PART2 即可，也可以稍微看看 PART4 学习如何优化自己的镜像以及容器的运行性能。

# 03 推荐学习路径

1. 首先可以在空闲的时间看看 [30 Min Into Docker](https://www.bilibili.com/video/BV14s4y1i7Vf/?spm_id_from=333.337.search-card.all.click&vd_source=0f7bf6e98597c64847b8d689f7a280c6) ，对其有个大概的了解
2. 然后在对其有一定的了解之后，跟着官方  [Get Start](https://docs.docker.com/get-started/) 文档以及 [BlakeHorse - Docker](https://www.bilibili.com/video/BV1HP4118797?spm_id_from=333.788.videopod.episodes&vd_source=0f7bf6e98597c64847b8d689f7a280c6&p=14) 中的第3，4集先练练手
3. 然后再观看黑马的完整视频来学习，这个过程可以参考官方文档以及参考书来学习。
4. 最后，根据所学的部署一个自己的网盘
	- cloudns, cloudflare, cloudreve

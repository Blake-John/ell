# 01 假期时间

2025年寒假持续时间为 `2025-1-13 - 2025-2-22` ，总计 `41` 天。其中主要分为一下几个阶段 : 

- `2025-1-13 - 2025-1-25` - 武术队年前集训时间，共 `13` 天
- `2025-1-26` - 坐车回老家
- `2025-1-27 - 2025-2-5` - 回家过年时间，共 `10` 天
- `2025-2-6` - 坐车回学校
- `2025-2-7 - 2025-2-22` - 武术队年后集训时间，共 `16` 天

## 1.1 集训时间

集训的时间安排为 : 

| 时间            | 安排     |
| ------------- | ------ |
| 9：00 - 11：00  | 上午训练阶段 |
| 15：30 - 18：00 | 下午训练阶段 |

因此，每日空余时间仍有 : 

| 时间            | 安排  |
| ------------- | --- |
| 6：30 - 8：30   | 空闲  |
| 11：00 - 14：30 | 空闲  |
| 18：00 - 22：30 | 空闲  |

## 1.2 回老家时间

回老家之后，每天的空余时间就变得很多了，但是也需要空出1-2天来走亲戚。除此之外，其余时间可以既有安排，但是无论怎么安排，有个最基本的要求： **在回老家期间仍然要保持规律作息，不能熬夜，要早睡早起** 。这既是为了能够保持身体的活性，不消耗年前集训的身体素质，更是为了年后的集训做准备。

规律作息包括：

- 每天运动1小时，可以分阶段，但需要达标
- 22：30前睡觉
- 每日健康饮食

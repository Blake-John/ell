# 2. 项目内容

- **分子坐标与距离计算**：
    - **坐标作用**：通过坐标确定分子中各点连接关系，计算点与点之间的距离。
    - **距离判断**：若两点距离小于 1.5 或 1.6 ，认为两点可连在一起。
      这是由于碳原子键大小以及碳氢键大小决定的
    - **坐标确定思路**：以原点为参照，通过旋转角度确定苯环各点坐标，可思考如何确定其他点的坐标。

> [!question] 
> 如何选择坐标原点及坐标的表示方法使得能够通过坐标信息得到原子连接方式？
> 或者说如何通过坐标正确计算出原子的连接方式？

- **凯库勒式表示方法**：
    - **表示形式**：以 **标注双键位置** 的形式表示凯库勒式，如指出几号原子间是双键。
    - **分子尝试**：对 a、b、c、d 等多种分子尝试确定其所有凯库勒式。

- **项目任务与步骤**：
    - **列出方程**：根据分子坐标 **列出线性方程组**，方程可用矩阵表示，明确矩阵中各系数。
    - **求解方程**：**解出方程所有可能的解**，a 取值只能为 2 或 4 。
    - **分工合作**：鼓励成员合作分工编写代码，可用 Python、MATLAB 等工具完成上述任务。

- **分子磁性与约束方程**：
    - **磁性原理**：微观状态下，分子中碳原子 **若有未配对电子则显示磁性**，如某些分子画凯库勒式时存在不满足 8 电子的碳原子。
    - **约束方程**：对每个碳原子依据其成键情况写出约束方程，**a 表示键（单键为 2，双键为 4），b 表示未成对电子（0 表示成对，1 表示未成对）**，复杂分子方程更复杂。
    - **解题建议**：先针对简单例子算出所有解，再尝试针对复杂分子依据坐标写出约束方程。

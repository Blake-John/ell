# 实践1

今天对苯环类似结构的能量进行了二次讨论与实践，主要考虑了总能量代表着什么，是否考虑C-H键的影响。

首先，总能量指的是分子的键能，而不是分子的总能量，即 **不需要减去基态原子的能量** 。今天对所有无磁分子的能量及占据数进行了首次拟合，采用 : 

$$E = \sum_{i}E_{0}\log{p_{i}}$$

其中 $E_{0} = 484.30501882848336$ ，是通过 275 个样本进行上述计算来拟合，并计算得到最小均方差的结果。但是，这个拟合在 **分子较大的时候拟合效果好**，**分子较小的时候拟合效果差** ，证明单纯的 $log$ 函数并不一定适用，而应该考虑其他的因素，如指数衰减等。通过上式计算的能量与实际能量的对比如下 : 

![[imgs/first_fit.png]]

接下来需要考虑不同分子量所带来的影响，因此 **选择同分异构体进行拟合探究**。

# 实践1总结

上述的实验实践与最开始的假设并不一致，最开始是通过类似期望来进行拟合，而上式没有引入自变量。需要重新进行拟合。并且计算误差的代码逻辑出错，计算的并非是总体样本的误差。

# 实践2

这一次修正了误差计算的代码，并通过如下公式进行拟合 : 

$$E = E_{0}\sum_{i}p_{i}\log{p_{i}}$$

但是结果仍然极差，如下图所示 : 

![[imgs/secon_fit.png]]

因此，现在基本确定了拟合的 "常量" $E_0$ 并非是一个常量，而是随着分子数/占据数的变化而变化。

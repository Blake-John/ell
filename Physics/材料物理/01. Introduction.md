### **Chapter 1: 引言与电子的经典理论**

---

#### **1.1 材料科学的历史视角**
这部分主要来自课件。它以材料的使用划分了人类文明的发展阶段，并引出材料科学的核心思想。

- **石器时代**：约200万年前，人类开始使用天然材料（石头、木材）。
- **青铜时代**：约5000年前，人类发明了合金（铜+锡）。青铜可以铸造成形，比纯铜更硬，且耐腐蚀。
- **铁器时代**：约3000年前延续至今，铁和钢的使用彻底改变了社会。
- **先进材料时代**：现在，我们进入了主动设计材料的阶段。理解材料“**加工-结构-性能-服役性能**”这四个要素及其相互关系至关重要。
    - **结构**（structure）决定**性能**（properties）。
    - **加工工艺**（processing）可以改变**结构**。
    - **服役性能**（performance）是材料在实际使用中的表现，由性能决定。
    - 课件中以氧化铝（Al₂O₃）的透明度为例，说明了不同加工工艺导致不同结构（单晶/多晶/含气孔），从而带来截然不同的光学服役性能。

- **材料的三大基本分类**：

| 材料类别    | 特点         | 键合方式              | 典型性能              |
| :------ | :--------- | :---------------- | :---------------- |
| **金属**  | 由金属元素组成    | 金属键               | 导电、导热好，有光泽，强韧，可变形 |
| **陶瓷**  | 金属与非金属的化合物 | 离子/共价键            | 绝缘，耐高温，坚硬但脆性大     |
| **聚合物** | 有机大分子      | 共价键(链内) + 次价键(链间) | 轻质，柔韧，易成型，不耐高温    |

- **材料选择的三大标准**：
    1.  **服役条件**：决定了所需材料的性能组合。
    2.  **材料在服役中的劣化行为**：如腐蚀、磨损等环境因素的影响。
    3.  **经济性**：从设计、加工到回收的全生命周期成本（“摇篮到坟墓”）。

#### **1.2 电子的经典粒子图像（教材第一章核心）**
这是整个电学性质课程的起跑线，我们首先尝试用一个简单的经典模型来解释固体中的电子行为。

- **1.2.1 基本模型：电子气**
    - 我们假设固体中有一个由原子核和芯电子组成的固定晶格，价电子可以在晶格间**自由移动**，就像气体分子一样，因此被称为“**自由电子气**”。
    - 在没有外电场时，电子进行无规则热运动，其平均动能与温度相关：$\dfrac{1}{2}mv_{th}^2 = \dfrac{3}{2}k_BT$。

- **1.2.2 欧姆定律的推导**
    - 施加电场 $\mathcal{E}$，电子获得一个定向的**漂移速度（drift velocity）** $v_D$，叠加在无规热运动上。
    - 电子在两次碰撞间的平均时间称为**弛豫时间 $\tau$**。
    - **迁移率 $\mu_e$**：$v_D = \dfrac{e\tau}{m}\mathcal{E} = \mu_e \mathcal{E}$。
    - **电流密度 $J$**：$J = n e v_D = \dfrac{n e^2 \tau}{m}\mathcal{E}$。
    - 对比欧姆定律 $J = \sigma \mathcal{E}$，我们得到电导率的核心公式：
    $$\sigma = \dfrac{n e^2 \tau}{m} = n e \mu_e$$
    电导率由**载流子浓度 `n`** 和**迁移率 $\mu_e$** 共同决定。

- **1.2.3 霍尔效应 (Hall Effect)**
    - 对一个通有电流的导体施加垂直于电流方向的磁场，电子会受到洛伦兹力的作用发生偏转。
    - 电荷在导体一侧积累，建立一个横向的**霍尔电场 $E_H$**。
    - 平衡时，霍尔电场力与洛伦兹力相等：$E_H = v_D B = R_H J B$。
    - **霍尔系数 $R_H$**：$R_H = \dfrac{1}{ne}$。
    - **重大意义**：测量 $R_H$ 可以直接得到载流子的**浓度 `n`** 和**符号**（带正电还是负电）。这是后来发现 **“空穴”（hole）** 这一概念的重要实验基础。

- **1.2.4 电磁波在固体中的传播与等离子体模型**
    - 将自由电子气模型代入麦克斯韦方程组，可以研究电磁波在导体中的传播。
    - 引入一个有效介电常数： $\varepsilon_{eff} = 1 - \dfrac{\omega_p^2}{\omega^2}$。
    - 其中，$\omega_p^2 = \dfrac{n e^2}{m \varepsilon_0}$ 被称为**等离子体频率**。
    - **物理图像**：
        - **当 $\omega > \omega_p$**：$\varepsilon_{eff} > 0$，波可以传播，材料表现出透明性。
        - **当 $\omega < \omega_p$**：$\varepsilon_{eff} < 0$，波矢为虚数，波在材料中迅速衰减（皮肤深度），发生全反射。材料不透明。
    - **成功应用**：这个模型完美地解释了为什么金属对于无线电波（$\omega < \omega_p$）是优良的反射体，而对于远紫外光（$\omega > \omega_p$，如碱金属的$\omega_p$在紫外区）则可变为透明。表1.1的理论计算值与实验值吻合得很好。

- **1.2.5 经典模型的成就与局限**
    - **辉煌的成就**：
        1.  成功推导出**欧姆定律**，并解释了电导率的微观意义。
        2.  解释了**霍尔效应**。
        3.  预言了金属对一个特定频率以上电磁波的**透明性**（等离子体模型）。
        4.  解释了金属的**高热导率**（Wiedemann-Franz定律），尽管常数吻合度不佳。
    - **根本性的失败**：
        **无法解释电子的热容**。根据经典的能量均分定理，自由电子应具有 $3/2 k_B$ 的单原子热容，但实验测得的电子热容非常小，仅在极低温度下才显现。这为后续量子理论的引入埋下了伏笔，说明经典模型是错的，波粒二象性是必需的。

# 5.1.1 研究对象

> 分析力学要研究的对象是什么？

分析力学要研究的是一个 **体系** ，是大量质点组成的体系。如果这个体系中的质点间 **存在着相互作用** ，以至于 **每一质点的运动** 都与 **其他质点的位置及运动有关** ，则这种体系叫做 *⌈力学体系⌋* ，在后面简称为 *⌈体系⌋* #def/分析力学 。

# 5.1.2 约束

## 1 定义

在一个力学体系中，常常存在着一些 **限制** 各质点运动的条件，我们把这些条件称之为 *⌈约束⌋* #def/分析力学 。

例如，一个质点被限制 **在某一线或面上运动而不能脱离** ，那么该线或面就称为约束，而这条线或这个面 **满足的方程** 就是 *⌈约束方程⌋* #def/分析力学 。

> [!example] 约束
> 约束对质点的限制条件通常可以表示为该质点 **坐标，速度和时间的方程** ，使得 **三个坐标(x, y, z)之间并不相互独立** 。 n 个质点，在空间中受到 k 个限制形成的力学体系，那么该体系就有 k 个表示这种约束的方程，因而独立坐标数为 **3n - k** 个，
> 
> 如果质点在一曲面 S 上运动，且该曲面的方程为 $f(x, y, z) = 0$ ，那么独立坐标的数目由 3 个减少为 2 个。因为已知 x, y 和时间 t 之间的函数，可以由约束方程得出 z 与 x, y 之间的关系，进而得出 z 与 t 之间的函数。
> 
> > 在上面的例子中，我们可以任意选择两个坐标为参考，并且这两个坐标之间相互独立，而不一定要选择 x, y 作为独立坐标。

## 2 分类

### a. 依时间分类

如果限制系统位置的约束 **不是时间 t 的函数** ，即 **约束方程中不显含时间 t** ，则这种约束为 *⌈稳定约束⌋* #def/分析力学 ；反之，如果约束 **是时间 t 的函数** ，即 **约束方程中显含时间 t** ，则这种约束是 *⌈不稳定约束⌋* #def/分析力学 。

> [!example] 
> $f(x, y, z) = 0$ 所满足的约束为 **稳定约束** ， $g(x, y, z, t) = 0$ 所满足的约束为 **不稳定约束**。
> 
> 例如，当一质点和一刚性杆连接时，
> 
> * 如果刚性杆的上端 **固定不动** ，取该点为原点，则约束方程为 $x^{2} + y^{2} + z^{2} = l^{2}$ ，这就是稳定约束
> * 如果刚性杆的上端 **沿水平直线以匀速** $c$ **运动**，则约束方程应为 $(x - ct)^{2} + y^{2} + z^{2} = l^{2}$ ，这就是不稳定约束

### b. 依质点运动状态分类

如果质点 **始终不能脱离该约束** ，那么这种约束叫 *⌈不可解约束⌋* #def/分析力学  ，如质点被限制在曲面 $f(x, y, z) = 0$ 或 $f(x, y, z, t) = 0$ 上而不能脱离该曲面，则该曲面为不可解约束。

如果质点在 **某一方向上可以脱离该约束** ，那么这种约束叫 *⌈可解约束⌋* #def/分析力学 ，如有这种约束 $f(x, y, z) \leq c$ ，质点虽然被限制在曲面 $f(x, y, z) = 0$ 上，但是可以沿着 $f(x, y, z) < c$ 的方向脱离曲面。

> [!note] 
> 不可解约束常以 **等式** 来表示，可解约束常以 **不等式** 来表示。

> [!example] 
> 例如，一个质点被一根 **柔软的轻绳** 固定住，那么该质点只能 **在绳长的范围内运动**，约束方程为 $x^{2} + y^{2} + z^{2} \leq l^{2}$ ；但如果该质点被一根 **刚性杆** 固定住，那么该质点只能在刚性杆画出的球面上运动，约束方程为 $x^{2} + y^{2} + z^{2} = l^{2}$ 。

### c. 依限制目标分类

如果一种约束 **只限制质点的空间位置，表现为坐标的函数** ，那么这种约束是 *⌈几何约束⌋* #def/分析力学 ，如 $f(x, y, z) = 0$ 。

如果一种约束 **既限制了质点的空间位置，又限制了质点的运动状态，即限制了质点的速度的大小** ，那么这种约束是 *⌈运动约束⌋* #def/分析力学 ，又称之为 *⌈微分约束⌋* #def/分析力学 ，如 $f(x, y, z; \dot{x}, \dot{y}, \dot{z}; t) = 0$ 。这种约束的方程中除了含有坐标外，还含有其微分项，因此称为微分约束。

几何约束又叫 *⌈完整约束⌋* #def/分析力学 ；而有时微分约束经过积分后可以转化为几何约束，有时不能，倘若它不能积分时，就被称为 *⌈不完整约束⌋* #def/分析力学 。除了不能被积分的微分约束外， **不能用等式表示的可解约束是另一种不完整约束** 。其他约束均为完整约束。

凡是 **只受完整约束** 的力学体系称为 *⌈完整系⌋* #def/分析力学 ，但凡 **含有不完整约束** 的力学体系称为 *⌈不完整系⌋* #def/分析力学 。

# 5.1.2 广义坐标

对于 n 个质点形成的力学体系，如果含有含有 k 个约束

$$f_{\alpha}(x, y, z, t) = 0 \quad (\alpha = 1, 2, \cdots, k), \tag{5.1.1}$$ ^2a68c8

那么，其 **独立坐标的数目就减少为 3n - k 个** 。在 **只受几何约束的条件下** ，这些 **独立坐标的数目** 就叫做力学体系的 *⌈自由度⌋* #def/分析力学 。

> 但对于微分约束来讲，自由度可能少于独立坐标的数目

如果令 $s = 3n - k$ ，那么我们就可以利用 [式 5.1.1](5.1%20约束与广义坐标.md#^2a68c8) 将这 3n 个不两两相互独立的坐标，转化为 s 个相互独立的参数 $q_{1}, q_{2}, \cdots, q_{s}$ ，即存在一变换使得

$$\left. \begin{array}{l}
x_{i} = x_{i}(q_{1}, q_{2}, \cdots, q_{s}, t) \\
y_{i} = y_{i}(q_{1}, q_{2}, \cdots, q_{s}, t) \\
z_{i} = z_{i}(q_{1}, q_{2}, \cdots, q_{s}, t) \\
\end{array}\right\} \quad (i = 1, 2, \cdots, n, s < 3n)
\tag{5.1.2}$$

^feb270

或

$$
\mathbf{r}_{i} = \mathbf{r}_{i}(q_{1}, q_{2}, \cdots, q_{s}, t) \quad (i = 1, 2, \cdots, n, s < 3n) \tag{5.1.3}
$$

式中 $q_{1}, q_{2}, \cdots, q_{s}$ 称为 *⌈拉格朗日广义坐标⌋* #def/分析力学 ，它可以代表你想要使用的各种物理量，不一定是坐标，可以是角度，面积，体积等等。

> [!example] 
> 一质点被限制在圆周 $x^{2} + y^{2} = R^{2}$ 上运动时，可以令 $x = R\cos \theta, y = R\sin \theta$ ，式中的 $\theta$ 是质点的 **极角** ，这就是一个 **广义坐标** 。
> 
> 在该问题中，质点原有两个坐标，但是由于被限制在圆周上运动，因此独立坐标的数目只有一个，自由度也只有一个，因此我们可以通过该约束将两个普通的坐标 $x, y$ 转化成广义坐标 $\theta$ 。$x^{2} + y^{2} = R^{2}$ 相当于 [式 5.1.1](5.1%20约束与广义坐标.md#^2a68c8) ，而 $x = R\cos \theta, y = R\sin \theta$ 相当于 [式 5.1.2](5.1%20约束与广义坐标.md#^feb270) 。

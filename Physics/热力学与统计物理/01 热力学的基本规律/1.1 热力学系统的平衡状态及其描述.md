# 01 热力学研究什么

#def/热统
- **研究对象** - 大量微观粒子组成的 **宏观物质系统** 
	- **外界** - 与系统发生相互作用的 **其他物体** 称为 「外界」

| 系统  | 物质交换 | 能量交换 |
| --- | ---- | ---- |
| 孤立系 | 无    | 无    |
| 闭系  | 无    | 有    |
| 开系  | 有    | 有    |

# 02 研究对象的特点

**热力学平衡态** - 一个孤立系统，无论其初态如何复杂，经过 **足够长的时间后** ，将会到达这样的状态——系统的 **各种宏观性质在长时间内不发生任何变化** 

与平衡态相关的一些性质：

- **弛豫时间** - 系统由其 **初始状态达到平衡状态** 所经历的时间。平衡态要求所研究的各种宏观性质都不随时间变化，相应地应 **取其中最长的驰豫时间作为系统的驰豫时间** 
- **热动平衡** - 组成系统的大量 **微观粒子仍不断运动** ，但是统计平均效果不变，是一种动态平衡
- **涨落** - 系统宏观物理量的数值仍会发生涨落，但极其微小，可以忽略不记
- **复杂平衡** - 平衡状态的概念 **不限于孤立系统** 。对于非孤立系，从原则上说，可以把系统与外界合起来看作一个 **复合的孤立系统**，然后根据孤立系统平衡状态的概念推断系统是否处在平衡状态。

# 03 如何描述平衡状态

我们选择几个宏观的物理量，这些自变量本身可以独立改变，其他宏观量又可以表达为他们的函数。成这些选定的物理量为 **状态参量** ，其函数为 **状态函数** 。

## 3.1 不同的参量

- **几何参量** 
- **力学参量** 
- **电磁参量** 
- **化学参量** 
- ...

## 3.2 不同的系统

- **简单系统** - 只通过体积 $V$ 和压强 $p$ 两个参量就可以确定的系统
- **相** - 系统中每一个 **均匀的部分**
	- **均匀系** - 各个部分的 **性质完全一样** ，因此也称为 **单相系** 
	- **复相系** - 整个系统不均匀，但是可以分为 **若干个均匀的部分** ，每一个部分都需要用参量来描述

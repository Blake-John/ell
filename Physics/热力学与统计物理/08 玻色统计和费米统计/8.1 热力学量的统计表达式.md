# 00 引言

本节的核心任务是：**在巨正则系综框架下，将宏观热力学量（N， U， Y， S， J等）表示为巨配分函数 $\varXi$ 的函数**。这是连接微观统计分布与宏观热力学性质的"翻译器"。

**历史背景与逻辑演进**：
- **第七章**基于**玻耳兹曼分布**（麦克斯韦-玻尔兹曼统计），适用于**非简并气体**（$n\lambda^3 \ll 1$）。该分布是在**孤立系**（微正则系综）条件下，通过**最概然分布法**导出的。
- **本节**要处理**简并气体**（玻色/费米气体），需采用**巨正则系综**（开系，可与源交换粒子与能量）。导出的分布是**平均分布**（而非最概然分布），其表达式与第七章相同，但**参量 $\alpha， \beta$ 的物理地位发生了根本改变**：
  - **最概然分布**：$\alpha， \beta$ 是拉格朗日乘子，由约束条件 $N， U$ 确定。
  - **平均分布**：$\alpha， \beta$ 是由**外界条件**（热源温度 $T$ 和粒子源化学势 $\mu$）确定的**已知参量**，而 $N， U$ 成为待求量。

**关键点**：这种视角转换使得我们可以先计算 $\ln\varXi(\alpha， \beta， y)$，再通过求导得到所有热力学量，逻辑上更简洁普适。

# 01 巨配分函数的构造与物理意义

> [[mics/玻色系统与费米系统|什么是玻色系统与费米系统？]] 

## 1.1 玻色系统的巨配分函数

对于单能级 $l$，其微观状态数（即该能级的配分函数）为：
$$\varXi_l = \sum_{a_l=0}^{\infty} e^{-(\alpha+\beta\varepsilon_l)a_l} = \dfrac{1}{1 - e^{-\alpha-\beta\varepsilon_l}}$$

**物理诠释**：这是对能级 $l$ 上所有可能的占据数 $a_l$（可以任意多）求和，权重因子 $e^{-(\alpha+\beta\varepsilon_l)a_l}$ 来自巨正则系综的权重 $e^{-\alpha N - \beta E}$。

对于整个系统，各能级独立，故巨配分函数是连乘：
$$\varXi = \prod_l \varXi_l^{\omega_l} = \prod_l (1 - e^{-\alpha-\beta\varepsilon_l})^{-\omega_l} \tag{8.1.2}$$

取对数后线性化，便于计算：
$$\ln\varXi = -\sum_l \omega_l \ln(1 - e^{-\alpha-\beta\varepsilon_l}) \tag{8.1.3}$$

## 1.2 费米系统的巨配分函数

对于费米子，泡利不相容原理限制每个量子态最多占据一个粒子（$a_l = 0， 1$）：
$$\varXi_l = \sum_{a_l=0}^{1} e^{-(\alpha+\beta\varepsilon_l)a_l} = 1 + e^{-\alpha-\beta\varepsilon_l}$$

因此：
$$\varXi = \prod_l (1 + e^{-\alpha-\beta\varepsilon_l})^{\omega_l} \tag{8.1.12}$$
$$\ln\varXi = \sum_l \omega_l \ln(1 + e^{-\alpha-\beta\varepsilon_l}) \tag{8.1.13}$$

**对比**：玻色系统的对数前是负号（源于 $(1-x)^{-1}$），费米系统是正号（源于 $1+x$），这决定了后续求导结果的符号差异。

# 03 热力学量的统计表达式推导（核心推导）

## 3.1 平均粒子数 $\bar{N}$

由巨正则系综理论：
$$\bar{N} = -\dfrac{\partial}{\partial\alpha}\ln\varXi$$

**详细推导**：
- 对玻色系统：$\ln\varXi = -\sum_l \omega_l \ln(1 - e^{-\alpha-\beta\varepsilon_l})$
- 求导：$-\dfrac{\partial}{\partial\alpha}\ln\varXi = -\sum_l \omega_l \cdot \dfrac{-e^{-\alpha-\beta\varepsilon_l}}{1 - e^{-\alpha-\beta\varepsilon_l}} = \sum_l \dfrac{\omega_l}{e^{\alpha+\beta\varepsilon_l} - 1}$
- 这正是玻色分布 $\sum_{l}a_l$，由 [[mics/玻色系统与费米系统#2.1 玻色-爱因斯坦分布|玻色分布平均粒子数]] 即可得

#### **2. 内能 $U$**
$$U = -\dfrac{\partial}{\partial\beta}\ln\varXi$$

**详细推导**：
- 对玻色系统：$-\dfrac{\partial}{\partial\beta}\ln\varXi = -\sum_l \omega_l \cdot \dfrac{-\varepsilon_l e^{-\alpha-\beta\varepsilon_l}}{1 - e^{-\alpha-\beta\varepsilon_l}} = \sum_l \dfrac{\omega_l \varepsilon_l}{e^{\alpha+\beta\varepsilon_l} - 1}$
- 这正是 $\sum_l \varepsilon_l a_l \tag{8.1.6}$ 

#### **3. 广义力 $Y$（以压强 $p$ 为例）**

广义力是 $\dfrac{\partial\varepsilon_l}{\partial y}$ 的统计平均：
$$Y = \sum_l \dfrac{\partial\varepsilon_l}{\partial y} a_l = -\dfrac{1}{\beta}\dfrac{\partial}{\partial y}\ln\varXi \tag{8.1.7}$$

**压强的特例**：
$$p = \dfrac{1}{\beta}\dfrac{\partial}{\partial V}\ln\varXi \tag{8.1.8}$$

**物理意义**：压强来源于粒子能量对体积的依赖（通过能级 $\varepsilon_l(V)$），体积变化导致能级移动，从而产生宏观力。

### **四、 热力学基本方程的统计推导（本节的精髓）**

#### **1. 构造恰当的全微分形式**

由(8.1.4)-(8.1.7)可得：
$$\beta\left(dU - Ydy + \dfrac{\alpha}{\beta}d\bar{N}\right) = -\beta d\left(\dfrac{\partial\ln\varXi}{\partial\beta}\right) + \dfrac{\partial\ln\varXi}{\partial y}dy - \alpha d\left(\dfrac{\partial\ln\varXi}{\partial\alpha}\right)$$

**关键步骤**：这里使用了 $U = -\partial_\beta\ln\varXi$，$Ydy = -\dfrac{1}{\beta}\partial_y\ln\varXi\，dy$，$\bar{N} = -\partial_\alpha\ln\varXi$

#### **2. 利用 $\ln\varXi$ 的全微分**

$\ln\varXi$ 是 $\alpha， \beta， y$ 的函数，其全微分为：
$$d\ln\varXi = \dfrac{\partial\ln\varXi}{\partial\alpha}d\alpha + \dfrac{\partial\ln\varXi}{\partial\beta}d\beta + \dfrac{\partial\ln\varXi}{\partial y}dy$$

**技巧性重组**：
通过加减项，可将右边重组为：
$$\beta\left(dU - Ydy + \dfrac{\alpha}{\beta}d\bar{N}\right) = d\left(\ln\varXi - \alpha\dfrac{\partial\ln\varXi}{\partial\alpha} - \beta\dfrac{\partial\ln\varXi}{\partial\beta}\right)$$

#### **3. 与热力学基本方程对应**

热力学中，对开系有：
$$dS = \dfrac{1}{T}(dU - Ydy - \mu d\bar{N})$$

将两式对比，可得**统计-热力学对应关系**：
$$\beta = \dfrac{1}{kT}， \quad \alpha = -\dfrac{\mu}{kT} \tag{8.1.9}$$

**物理意义**：
- $\beta$ 是能量涨落的"damping factor"，其倒数正比于温度
- $\alpha$ 与化学势相关，负号保证 $\mu < \varepsilon_0$（玻色系统基态能量为零时 $\mu < 0$）

#### **4. 熵的统计表达式**

积分得到：
$$S = k\left(\ln\varXi + \alpha\bar{N} + \beta U\right) \tag{8.1.10}$$

**玻耳兹曼关系的再证明**：
将 $\ln\varXi$ 的具体形式代入(8.1.10)，利用斯特林近似，可证明：
$$S = k\ln\Omega$$
其中 $\Omega$ 是系统总微观状态数。这说明**玻耳兹曼关系在量子统计中依然成立**，且具有更深刻的内涵。

---

### **五、 特性函数与巨热力势**

#### **1. $\ln\varXi$ 作为特性函数**

- **自然变量**：$\alpha， \beta， y$（即 $T， V， \mu$）
- **完备性**：由 $\ln\varXi$ 可导出所有热力学量，故为特性函数

#### **2. 巨热力势 $J$**

定义 $J = U - TS - \bar{N}\mu$，将(8.1.10)代入可得：
$$J = -kT\ln\varXi \tag{8.1.14}$$

**意义**：
- 在热力学中，$J(T，V，\mu)$ 是以 $T，V，\mu$ 为自然变量的特性函数
- 统计物理给出了其明确的微观表达式，架起了统计与热力学的桥梁

---

### **六、 从 $(\alpha，\beta，V)$ 到 $(N，T，V)$ 的转换**

实际应用中，给定的是 $N，T，V$ 而非 $\alpha，\beta，V$。求解步骤：

1. **计算**：先求得 $\ln\varXi(\alpha，T，V)$
2. **反解**：由 $\bar{N}(\alpha，T，V) = N$ 反解出 $\alpha(N，T，V)$，这通常需要数值或近似方法
3. **代入**：将 $\alpha(N，T，V)$ 代回各表达式，得到热力学量作为 $N，T，V$ 的函数

**近似策略**：
- **弱简并**（8.2节）：$e^{-\alpha} \ll 1$，级数展开
- **强简并**：$e^{-\alpha} \gg 1$，索末菲展开（8.5节）
- **临界区域**（BEC）：需分离基态贡献（8.3节）

### **七、 学习要点与易错警示**

#### **1. 符号约定**
- 公式中 $\pm$ 号：**上方"+"对应费米，下方"-"对应玻色**
- 对数中玻色为负、费米为正，但求导后结果形式统一

#### **2. 平均分布 vs 最概然分布**
- **概念区别**：最概然分布是出现概率最大的分布；平均分布是系综平均
- **实践等价**：对于宏观系统，涨落极小，两者数值差异可忽略
- **理论进步**：平均分布基于系综理论，逻辑基础更坚实（见第九章）

#### **3. 全微分重组的技巧**
这是推导的关键，本质是**勒让德变换**的统计体现，将变量从 $(\alpha，\beta)$ 转换到 $(\bar{N}，U)$

#### **4. 积分因子的识别**
统计表达式 $\beta(dU - Ydy + \dfrac{\alpha}{\beta}d\bar{N})$ 与热力学 $dS = \dfrac{1}{T}(dU - Ydy - \mu d\bar{N})$ 的对应是**统计力学与热力学等价的证明**，是本节最深刻的结论。

---

### **八、 总结：理论框架的建立**

本节完成了**从微观到宏观的统计力学范式**：
1. **微观输入**：粒子能级结构 $\varepsilon_l(y)$ 和简并度 $\omega_l$
2. **统计核心**：构造巨配分函数 $\ln\varXi(\alpha，\beta，y)$
3. **宏观输出**：通过求导运算获得全部热力学量
4. **理论自洽**：证明统计表达式满足热力学基本方程，且熵满足玻耳兹曼关系

**对后续章节的奠基作用**：
- 8.2节弱简并：对 $\ln\varXi$ 作级数展开
- 8.3节BEC：需分离基态贡献，因积分会丢失 $\varepsilon=0$ 项
- 8.4节光子气体：$\alpha=0$ 的特殊情况，简化计算
- 8.5节自由电子：强简并下索末菲展开 $\ln\varXi$

这一框架是量子统计力学的**通用语言**，掌握它才能理解后续具体应用的本质。

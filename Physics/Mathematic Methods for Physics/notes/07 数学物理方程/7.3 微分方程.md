# 7.3.1 线性二阶偏微分方程

## 1. 定义

将所有自变量（**空间坐标及时间坐标**）记为 $x_{1}, x_{2}, \cdots, x_{n}$，则 **二阶偏微分方程**可表示为：

$$
\sum_{j=1}^n \sum_{i=1}^n a_{ij} u_{x_i x_j} + \sum_{i=1}^n b_i u_{x_i} + c u + f = 0
$$

其中 $a_{ij}, b_i, c, f$ 只是 $x_{1}, x_{2}, \cdots, x_{n}$ 的函数，则该方程为 **线性方程**。

## 2. 一维展开

在一维方向上，二阶偏微分方程可以展开为：

$$
a_{11} u_{xx} + a_{12} u_{xt} + a_{21} u_{tx} + a_{22} u_{tt} + b_1 u_x + b_2 u_t + c u + f = 0
$$

例如：

1. 弦振动方程：

$$
\rho \dfrac{\partial^2 u}{\partial t^2} - T \dfrac{\partial^2 u}{\partial x^2} = 0
$$

其中T为关于x的函数。

2. 热传导方程：

$$
c ρ \dfrac{\partial u}{\partial t} - k_{0} \dfrac{\partial^2 u}{\partial x^2} = 0
$$

3. 波动方程：

$$
\rho \dfrac{\partial^2 u}{\partial t^2} - T \nabla^{2}u = 0
$$

其中T为关于x的函数。

## 3. 线性算符

满足如下关系式的算符 $\mathcal{L}$ 为 **⌈线性算符⌋** #def/数物方法/数学物理方程 ：

$$
\mathcal{L}[C_1 u_1 + C_2 u_2] = C_1 \mathcal{L}[u_1] + C_2 \mathcal{L}[u_2]
$$

其中 $C_{1}, C_{2}$为任意常数。

例如：

1. 偏导数算符：

$$
\dfrac{\partial}{\partial x} (C_1 u_1 + C_2 u_2) = C_1 \dfrac{\partial u_1}{\partial x} + C_2 \dfrac{\partial u_2}{\partial x}
$$

因此， $\dfrac{\partial}{\partial x}$ 为线性算符。

2. 乘以函数 $a(x)$ 的偏导数算符：

$$
a(x) \dfrac{\partial}{\partial x} (C_1 u_1 + C_2 u_2) = a(x) C_1 \dfrac{\partial u_1}{\partial x} + a(x) C_2 \dfrac{\partial u_2}{\partial x}
$$

因此，$a(x) \dfrac{\partial}{\partial x}$ 为线性算符。

## 4. 线性方程

一个 **线性算符** 作用到 **待求的函数** 上并组合成方程，则为 **⌈线性方程⌋** #def/数物方法/数学物理方程 。即：

$$
\mathcal{L}[u(x, y, t)] = f(x, y, t)
$$

若 $\mathcal{L}[u(x, y, t)] = 0$，则该方程称为 **⌈线性齐次方程⌋** #def/数物方法/数学物理方程 。

若上述方程有两个解 $\mathcal{L}[u_{1}] = 0，\mathcal{L}[u_{2}] = 0$，由于 $\mathcal{L}$ 为线性算符，则：

$$
\mathcal{L}[C_1 u_1 + C_2 u_2] = C_1 \mathcal{L}[u_1] + C_2 \mathcal{L}[u_2] = 0
$$

即 $C_{1}u_{1} + C_{2}u_{2}$ 也为 $\mathcal{L}[u] = 0$ 的解。

由于 $C_{1}, C_{2}$ 为任意常数，故 $\mathcal{L}[u] = 0$ 有无穷多个解。

拉普拉斯变换最重要的一个应用就是 **用于解微分方程** 。

# 6.3.1 电路方程求解

## 1. RL电路

对于一个RL电路，我们通过以下方程来表述 **零状态响应** ：

$$\begin{cases}
L \dfrac{\mathrm{d}i_{L}}{\mathrm{d}t} + Ri_{L} = E_{0}\sin \omega t \\
i_{L}(0) = 0
\end{cases} \tag{6.3.1}$$

^44e9f7

若在时域上解这个方程，则我们需要先求出 **齐次方程的解** ，再求出 **非齐次方程的通解** ，最后再带入初始条件，过程较为繁复。

但是，我们可以通过转换到 **复频域** 来求解方程，这样能大大减少计算量。首先，对 $i_L$ 进行拉普拉斯变换，则有 $\mathcal{L}[i_{L}] = I$ ，由导数定理，可以得 $\mathcal{L}[\dfrac{\mathrm{d}i_{L}}{\mathrm{d}t}] = p I - i_{L}(0)$ ，由线性性，可得 $\mathcal{L}[Ri_{L}] = RI$ ，以及 $\mathcal{L}[E_{0}\sin \omega t] = E_{0} \dfrac{\omega}{p^{2} + \omega^{2}}$ ，因此， [[6.3 拉普拉斯变换的应用#^44e9f7|方程 6.3.1]] 变为 : 

$$
pIL + RI = E_{0} \dfrac{\omega}{p^{2} + \omega^{2}} \tag{6.3.2}
$$

因此，我们得到 : 

$$
I = E_{0} \dfrac{\omega}{p^{2} + \omega^{2}} \dfrac{1}{pL + R}
$$

设 

$$\begin{align}
I &= \dfrac{E_{0}}{L} \dfrac{\omega}{p^{2} + \omega^{2}} \dfrac{1}{p + \frac{R}{L}} \\
&= \dfrac{K_{1}}{p + \frac{R}{L}} + \dfrac{K_{2}}{p + i\omega} + \dfrac{K_{3}}{p - i\omega}
\end{align}$$

因此，我们有 : 

$$
\begin{cases}
K_{1} = \lim_{ p \to -\frac{R}{L} } \left( p + \dfrac{R}{L} \right)I = \dfrac{E_{0}}{L} \dfrac{\omega}{\left( \frac{R}{L} \right)^{2} + \omega^{2}} = \dfrac{\omega LE_{0}}{R^{2} + (\omega L)^{2}} \\

K_{2} = \lim_{ p \to -iw } \left( p + iw \right)I = \dfrac{E_{0}}{L} \dfrac{1}{-i\omega + \frac{R}{L}} \dfrac{\omega}{-i\omega - i\omega} = \dfrac{E_{0}}{L} \dfrac{L}{iwL - R} \dfrac{1}{2i} = -\dfrac{E_{0}}{2} \dfrac{\omega L - iR}{(\omega L)^{2} + R^{2}} \\

K_{3} = \lim_{ p \to iw } \left( p - iw \right)I = \dfrac{E_{0}}{L} \dfrac{1}{i\omega + \frac{R}{L}} \dfrac{\omega}{i\omega + i\omega} = \dfrac{E_{0}}{L} \dfrac{L}{iwL + R} \dfrac{1}{2i} = - \dfrac{E_{0}}{2} \dfrac{\omega L + iR}{(\omega L)^{2} + R^{2}}
\end{cases}
$$

即有

$$
\begin{align}
I &= \dfrac{\omega LE_{0}}{R^{2} + (\omega L)^{2}} \dfrac{1}{p + \frac{R}{L}}
- \dfrac{E_{0}}{2} \dfrac{\omega L - iR}{(\omega L)^{2} + R^{2}} \dfrac{1}{p + i\omega}
- \dfrac{E_{0}}{2} \dfrac{\omega L + iR}{(\omega L)^{2} + R^{2}} \dfrac{1}{p - i\omega} \\

&= \dfrac{\omega LE_{0}}{R^{2} + (\omega L)^{2}} \dfrac{1}{p + \frac{R}{L}} - \dfrac{E_{0}}{2} \dfrac{\omega L - iR}{(\omega L)^{2} + R^{2}} \dfrac{p - i\omega}{p^{2} + \omega^{2}} - \dfrac{E_{0}}{2} \dfrac{\omega L + iR}{(\omega L)^{2} + R^{2}} \dfrac{p + i\omega}{p^{2} + \omega^{2}}  \\

&= \dfrac{\omega LE_{0}}{R^{2} + (\omega L)^{2}} \dfrac{1}{p + \frac{R}{L}} - \dfrac{E_{0}}{2} \dfrac{2\omega L}{(\omega L)^{2} + R^{2}} \dfrac{p}{p^{2} + \omega^{2}} + \dfrac{E_{0}}{2} \dfrac{2R}{(\omega L)^{2} + R^{2}} \dfrac{\omega}{p^{2} + \omega^{2}}
\end{align}
$$


因此，我们通过反演得到 $i_{L}$ 的解：

$$i_{L}(t) = \dfrac{\omega LE_{0}}{R^{2} + (\omega L)^{2}}e^{ -\frac{R}{L}t } 

- \dfrac{\omega LE_{0}}{(\omega L)^{2} + R^{2}}\cos \omega t + \dfrac{RE_{0}}{(\omega L)^{2} + R^{2}}\sin \omega t
$$

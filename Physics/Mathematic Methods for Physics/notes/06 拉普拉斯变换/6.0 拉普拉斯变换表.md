# 常用拉普拉斯变换表

|             时间域函数 $f(t)$              |              复频域函数 $\bar{f}(p)$              |
| :-----------------------------------: | :------------------------------------------: |
|                  $1$                  |                $\dfrac{1}{p}$                |
|                  $A$                  |                $\dfrac{A}{p}$                |
|                  $t$                  |               $\dfrac{1}{p^2}$               |
|                 $t^n$                 |            $\dfrac{n!}{p^{n+1}}$             |
|    $\dfrac{1}{(n - 1)!}t^{n - 1}$     |              $\dfrac{1}{p^{n}}$              |
|               $e^{at}$                |              $\dfrac{1}{p - a}$              |
|               $Ae^{at}$               |              $\dfrac{A}{p - a}$              |
|              $t e^{at}$               |            $\dfrac{1}{(p - a)^2}$            |
|             $t^n e^{at}$              |         $\dfrac{n!}{(p - a)^{n+1}}$          |
| $\dfrac{1}{(n - 1)!}t^{n - 1} e^{at}$ |           $\dfrac{1}{(p - a)^{n}}$           |
|           $\sin(\omega t)$            |       $\dfrac{\omega}{p^2 + \omega^2}$       |
|           $\cos(\omega t)$            |         $\dfrac{p}{p^2 + \omega^2}$          |
|        $e^{at} \sin(\omega t)$        |    $\dfrac{\omega}{(p - a)^2 + \omega^2}$    |
|        $e^{at} \cos(\omega t)$        |    $\dfrac{p - a}{(p - a)^2 + \omega^2}$     |
|              $\sinh(at)$              |            $\dfrac{a}{p^2 - a^2}$            |
|              $\cosh(at)$              |            $\dfrac{p}{p^2 - a^2}$            |
|          $t \sin(\omega t)$           |   $\dfrac{2 \omega p}{(p^2 + \omega^2)^2}$   |
|          $t \cos(\omega t)$           | $\dfrac{p^2 - \omega^2}{(p^2 + \omega^2)^2}$ |

以上表格列出了一些常用函数的拉普拉斯变换对，方便在实际应用中快速查阅和使用。

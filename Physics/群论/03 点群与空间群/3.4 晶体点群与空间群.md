# 3.4 晶体点群与空间群

## 一、什么是晶体

晶体由全同的、由原子的集合构成的结构单元组成，这个结构单元在三维空间中可以**无限地**重复。晶体具有**转动（含转动反演）**与**平移**两种对称性。

**晶体点群**：忽略了平移对称性，能够将晶体变到其本身的转动与转动反演对称性的集合。在晶体点群的所有对称操作中，晶体有一个点是不动的。

## 二、晶体制约定理

**定理 3.4**：设 G 是晶体点群，则 G 中转动元素只能是 C₁, C₂, C₃, C₄, C₆，所有转动反演元素只能是 I, IC₂, IC₃, IC₄, IC₆。

### 证明

晶格 L 由三个线性无关向量 a₁, a₂, a₃ 的整数线性组合构成。对任意 g ∈ G：

$$g \boldsymbol{a}_i = \sum_{j=1}^{3} c_{ji} \boldsymbol{a}_j \tag{3.14}$$

**左边**：g aᵢ 必须在晶格上，因此 cⱼᵢ 必为整数。

**右边**：以 a₁, a₂, a₃ 为基，系数矩阵 C(g) 是群 G 的表示矩阵。这个表示与用 i, j, k 为基的表示差一个相似变换，特征标不变：

$$\operatorname{tr}(C(g)) = \pm(1 + 2\cos\Psi)$$

其中 Ψ 是转角。

由于 cᵢᵢ 为整数，tr(C(g)) = Σ cᵢᵢ 也为整数，所以 ±(1 + 2cosΨ) 必为整数。因此 Ψ 只能是 0, π/3, π/2, 2π/3, π。

纯转动：C₁(E), C₂, C₃, C₄, C₆
转动反演：I, IC₂, IC₃, IC₄, IC₆

> [!important] 为什么晶体中没有五阶轴？
> Ψ = 2π/5 时，1 + 2cos(2π/5) ≈ 1.618（黄金比例），不是整数。晶格的平移周期性禁止了五阶转动。分子中允许五阶轴（如环戊烷），但晶体中不允许。

## 三、晶体中允许的 32 种点群

### 第一类（11 个）

C₁, C₂, C₃, C₄, C₆, D₂, D₃, D₄, D₆, O, T

（Y 被排除，因为它有五阶轴）

### 第二类（21 个）

| 类型 | 点群 | 数量 | 说明 |
|------|------|------|------|
| S_{2n} | S₂, S₄, S₆ | 3 | S_{4n+2} 中只 S₂, S₆ 可存在；S_{4n} 中只 S₄ 可存在 |
| C_{nv} | C_{2v}, C_{3v}, C_{4v}, C_{6v} | 4 | 与 D_n 同构 |
| C_{nh} | C_{1h}, C_{2h}, C_{3h}, C_{4h}, C_{6h} | 5 | C_{2nh} 贡献 C_{2h}, C_{4h}, C_{6h}；C_{(2n+1)h} 贡献 C_{1h}, C_{3h} |
| D_{nh} | D_{2h}, D_{3h}, D_{4h}, D_{6h} | 4 | D_{(2n+1)h} 贡献 D_{3h}；D_{2nh} 贡献 D_{2h}, D_{4h}, D_{6h} |
| D_{nd} | D_{2d}, D_{3d} | 2 | D_{2nd} 贡献 D_{2d}；D_{(2n+1)d} 贡献 D_{3d} |
| 其他 | T_h, T_d, O_h | 3 | |

**总计：11 + 21 = 32 种晶体点群**

## 四、七大晶系

32 种点群按对称元素的相似性归入七个晶系。

> [!tip] 晶系分类的历史背景
> 最早研究晶体的学者没有 X 射线衍射，只能靠宏观特征分类。比如立方晶系的晶体（如 NaCl），x、y、z 三个方向等价，宏观上表现为各向同性。如果沿 z 轴拉伸，破坏了立方对称性但保留了四阶轴，就变成了四方晶系。

| 晶系 | 对称元素共性 | 晶胞参数 | 点群数 | 全面对称群 |
|------|-------------|----------|--------|-----------|
| **三斜** | 仅一阶轴 | a≠b≠c, α≠β≠γ≠90° | 2 | S₂ |
| **单斜** | 一个二阶轴 | a≠b≠c, α=β=90°, γ≠90° | 3 | C_{2h} |
| **正交** | 三个相互垂直的二阶轴 | a≠b≠c, α=β=γ=90° | 3 | D_{2h} |
| **四方** | 一个四阶轴 | a=b≠c, α=β=γ=90° | 7 | D_{4h} |
| **三方** | 一个三阶轴 | a=b≠c, α=β=90°, γ=120° 或 a=b=c, α=β=γ≠90° | 5 | D_{3d} |
| **六角** | 一个六阶轴 | a=b≠c, α=β=90°, γ=120° | 7 | D_{6h} |
| **立方** | 四个三阶轴 | a=b=c, α=β=γ=90° | 5 | O_h |

## 五、Bravais 格子（14 种）

**晶系**基于点群对称性，**晶格系统**基于空间群对称性。两者大部分重合，但三方/六角有交叉。

通过在晶胞的体心、面心等高对称点添加重复单元（要求不改变点群对称性且不能进一步简化），得到 14 种 Bravais 格子：

| 晶格系统 | Bravais 格子 | 符号 |
|----------|-------------|------|
| 三斜 | 三斜简单 | aP |
| 单斜 | 单斜简单、单侧心 | mP, mA |
| 正交 | 正交简单、体心、面心、全面心 | oP, oI, oF, oA |
| 四方 | 四方简单、四方体心 | tP, tI |
| 菱方 | 菱方简单 | hR |
| 六角 | 六角简单 | hP |
| 立方 | 立方简单、体心、面心 | cP, cI, cF |

> [!important] 晶系与晶格系统的区别
> - **三方晶系**的晶体：一部分具有菱方格子（hR），一部分具有六角格子（hP）
> - **六角晶系**的晶体：全部具有六角格子（hP）
> - 晶格系统中的"菱方"≠晶系中的"三方"。看到"菱方"想到 Bravais 格子分类，看到"三方"想到晶系分类。

### 晶系与晶格系统的相互关系

| 晶系 | 点群 | Bravais 格子 | 晶格系统 |
|------|------|-------------|----------|
| 三斜 | 2 (S₂, C₁) | 1 | 三斜 |
| 单斜 | 3 (C_{2h}, C₂, C_{1h}) | 2 | 单斜 |
| 正交 | 3 (D_{2h}, D₂, C_{2v}) | 4 | 正交 |
| 四方 | 7 (D_{4h}, C₄, S₄, D₄, C_{4v}, C_{4h}, D_{2d}) | 2 | 四方 |
| 三方 | 5 (D_{3d}, S₆, C₃, C_{3v}, D₃) | 1 + 1 | 菱方 + 六角 |
| 六角 | 7 (D_{6h}, C₆, C_{3h}, C_{6h}, C_{6v}, D₆, D_{3h}) | 1 | 六角 |
| 立方 | 5 (O_h, T, O, T_h, T_d) | 3 | 立方 |
| **共 7 种** | **共 32 种** | **共 14 种** | **共 7 种** |

## 六、从点群到空间群

### 空间群的定义

点群只考虑了转动对称性，对称操作为 R。空间群同时考虑了平移不变性，对称操作为 **{R|t}**，晶体在 {R|t} 操作下回到自身。

### 简单空间群（73 种）

最简单的组合方式：**点群 × Bravais 格子**。

以立方晶系为例：允许 T, O, T_h, T_d, O_h 五种点群，简单、体心、面心三种格子 → 3 × 5 = 15 种空间群。

对所有晶系做类似计算，初步得到 66 种。但某些 Bravais 格子与点群的结合方式不止一种（如正交晶系的底心格子与 C_{2v} 结合时，底心和侧面面心给出不同空间群），所以实际简单空间群是 **73 种**。

> [!note] 简单空间群的特征
> - 对所有 R，取 t = 0，{R|0} 都是空间群中的元素
> - 平移矢量 t 是平移周期性最小重复单元的整数倍

### 非简单空间群（157 种）

当 t 不是最小重复单元的整数倍，而是其**分数倍**时，出现两种新的对称操作：

**(1) 螺旋轴（screw axis）**：绕某轴转一定角度后，再沿该轴做一个平移，系统回到原来状态。

> 例如：纯平移周期为 a，但转动 60° 再做 a/2 的平移也是对称操作。对应的 {R|t} 中 R 不能脱离 t 单独存在。

**(2) 滑移面（slide plane）**：对某平面做反射后，再平移 t/m，系统回到原来状态。

> 例如：沿 x 轴平移 a 是对称操作，平移 a/2 不是，但它与镜面反射结合就是对称操作。

非简单空间群共有 **157 种**。

### 空间群总数

$$73 + 157 = 230 \text{ 种}$$

> [!history] 空间群的历史
> - **1879 年**：德国数学家 Sohncke 推出 66 种空间群（有重复）
> - **1891 年**：Sohncke 发表专著
> - **1891-1892 年**：德国人 Schoenflies 和俄国人 Fedorov 分别独立发表 230 种空间群的完整理论

### 实用工具

- **Bilbao Crystallographic Server**：输入空间群序号（如 122, 150），即可查看所有对称操作 {R|t}，还可以查看 Brillouin 区形状等信息。最近还增加了双群和磁空间群的内容。
- **英国 UCL 网站**：http://img.chem.ucl.ac.uk/sgp/large/sgp.htm，类似功能。

> [!tip] 从空间群确定晶体结构
> 很多文献给晶体结构时只给出不等价原子的位置和空间群。对每个原子，用空间群对称操作操作一遍，如果得到的原子已出现就略过，否则保存，从而得到完整的晶体结构。Material Studio 等软件也提供类似功能。

## 七、Hermann-Mauguin 符号（国际符号）

分子对称性用 Schoenflies 符号，晶体结构用 **Hermann-Mauguin 符号**，因为后者能更好地描述平移对称性。

**基本符号**：
- 纯转动轴：1, 2, 3, 4, 6
- 转动反演轴：$\overline{1}$, $\overline{2}$, $\overline{3}$, $\overline{4}$, $\overline{6}$

**与 Schoenflies 的对应**：

| Schoenflies | Hermann-Mauguin | 说明 |
|-------------|----------------|------|
| S₁ | $\overline{2}$ = m | 反射面 |
| S₂ | $\overline{1}$ | 反演中心 |
| S₃ | $\overline{6}$ | |
| S₄ | $\overline{4}$ | |
| S₆ | $\overline{3}$ | |
| C_n | n | |
| D₂ | 222 | |
| D₃ | 32 | |
| D₄ | 422 | |
| C_{2v} | mm2 | |
| C_{3v} | 3m | |
| C_{4v} | 4mm | |
| D_{2h} | 2/m 2/m 2/m | |
| D_{4h} | 4/m mm | |
| O_h | 4/m $\overline{3}$ 2/m | |
| T | 23 | |

## 八、总结：层次结构

```
O(3)（无限群）
  └── 点群（有限子群）
        ├── 第一类（纯转动）：5 种（C_n, D_n, T, O, Y）
        └── 第二类（含反射/反演）：9 种标记
              │
              ↓ 晶体限制（晶体制约定理）
              │
        32 种晶体点群
              │
              ↓ 按对称元素相似性分组
              │
        7 种晶系
              │
              ↓ 加入平移周期性
              │
        14 种 Bravais 格子（7 种晶格系统）
              │
              ↓ 点群 × 晶格 + 螺旋轴/滑移面
              │
        230 种空间群（73 简单 + 157 非简单）
```

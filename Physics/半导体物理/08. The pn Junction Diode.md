# 08. The pn Junction Diode

## 8.1 pn Junction Current

### 8.1.1 Qualitative Description of Charge Flow in a pn Junction

**核心问题：为什么加正向偏压就有电流？**

通过能带图理解三种偏置状态：

**零偏压（热平衡）：**
- n区导带有大量电子，p区价带有大量空穴
- 内建势垒 $V_{bi}$ 挡住载流子：n区电子要进入p区必须翻越能量"山峰"，p区空穴同理
- 势垒精确平衡扩散趋势和电场力，净电流为零

**反向偏压（n接正、p接负）：**
- 势垒从 $V_{bi}$ 升高到 $(V_{bi} + V_R)$
- 载流子被更牢固地"锁"在各自区域
- 几乎没有电流（仅有极小的反向饱和电流）

**正向偏压（p接正、n接负）：**
- 势垒从 $V_{bi}$ 降低到 $(V_{bi} - V_a)$
- 耗尽层内电场减弱
- n区电子扩散穿过耗尽层进入p区 → 成为p区的**少数载流子（过剩电子）**
- p区空穴扩散穿过耗尽层进入n区 → 成为n区的**少数载流子（过剩空穴）**
- 这就是**少数载流子注入**（minority carrier injection）
- 注入的少数载流子在对方区域经历**扩散和复合**（第6章的双极输运过程），扩散运动产生扩散电流

**直观类比：** 势垒＝水坝。零偏压时水坝高度刚好挡住两边的水；反向偏压是加高水坝；正向偏压是降低水坝，水从高水位一侧涌向低水位一侧。

### 8.1.2 Ideal Current–Voltage Relationship

推导理想I-V关系的**四条核心假设**：

| 假设 | 内容 | 含义 |
|------|------|------|
| **假设1** | 突变耗尽层近似 | 耗尽层有明确边界，耗尽层外半导体电中性 |
| **假设2** | 麦克斯韦-玻尔兹曼统计 | 非简并半导体，可用玻尔兹曼分布近似 |
| **假设3** | 低注入与完全电离 | 注入的少数载流子浓度远小于多数载流子浓度；所有掺杂原子电离，$n_{n0} \approx N_d$，$p_{p0} \approx N_a$ |
| **假设4a** | 总电流恒定 | 整个pn结构中总电流为常数（无电荷积累） |
| **假设4b** | 电流连续 | 电子电流和空穴电流各自为位置的连续函数 |
| **假设4c** | 耗尽层内电流恒定 | 耗尽层内无复合无产生，电子/空穴电流各自为常数 |

**关键符号表：**

| 符号 | 含义 |
|------|------|
| $N_a$ | p区受主浓度 |
| $N_d$ | n区施主浓度 |
| $n_{n0} = N_d$ | n区热平衡多数载流子（电子）浓度 |
| $p_{p0} = N_a$ | p区热平衡多数载流子（空穴）浓度 |
| $n_{p0} = n_i^2 / N_a$ | p区热平衡少数载流子（电子）浓度 |
| $p_{n0} = n_i^2 / N_d$ | n区热平衡少数载流子（空穴）浓度 |
| $\delta n_p = n_p - n_{p0}$ | p区过剩少数载流子（电子）浓度 |
| $\delta p_n = p_n - p_{n0}$ | n区过剩少数载流子（空穴）浓度 |
| $L_n = \sqrt{D_n \tau_{n0}}$ | 电子扩散长度 |
| $L_p = \sqrt{D_p \tau_{p0}}$ | 空穴扩散长度 |

### 8.1.3 Boundary Conditions

**目标：确定耗尽层边缘的少数载流子浓度。**

**Step 1：从内建势垒出发**

$$V_{bi} = V_t \ln\left(\frac{N_a N_d}{n_i^2}\right)$$

其中 $V_t = kT/e$ 是热电压（室温下约 25.9 mV）。

变形得到：

$$\frac{n_i^2}{N_a N_d} = \exp\left(\frac{-eV_{bi}}{kT}\right) \tag{8.1}$$

**Step 2：建立热平衡关系**

利用完全电离假设：
- $n_{n0} \approx N_d \tag{8.2}$
- $n_{p0} \approx n_i^2 / N_a \tag{8.3}$

代入(8.1)：

$$n_{p0} = n_{n0} \exp\left(\frac{-eV_{bi}}{kT}\right) \tag{8.4}$$

**物理意义：** p区边缘少数载流子电子浓度等于n区多数载流子电子浓度乘以玻尔兹曼因子 $\exp(-eV_{bi}/kT)$，即"翻越势垒的概率"。

**Step 3：加上正向偏压**

正向偏压 $V_a$ 将势垒从 $V_{bi}$ 降低到 $(V_{bi} - V_a)$，代入(8.4)：

$$n_p = n_{n0} \exp\left(\frac{-e(V_{bi} - V_a)}{kT}\right)$$

展开指数：

$$n_p = n_{n0} \exp\left(\frac{-eV_{bi}}{kT}\right) \exp\left(\frac{+eV_a}{kT}\right)$$

利用(8.4)替换第一项的乘积：

$$\boxed{n_p = n_{p0} \exp\left(\frac{eV_a}{kT}\right)} \tag{8.6}$$

**过剩载流子浓度：**

$$\delta n_p = n_p - n_{p0} = n_{p0}\left[\exp\left(\frac{eV_a}{kT}\right) - 1\right]$$

同理，n区耗尽层边缘：

$$p_n(x_n) = p_{n0} \exp\left(\frac{eV_a}{kT}\right)$$

$$\delta p_n = p_{n0}\left[\exp\left(\frac{eV_a}{kT}\right) - 1\right]$$

**物理意义：** 正向偏压降低势垒，注入的少数载流子浓度指数增加。这就是二极管I-V特性指数关系的起源。

### 8.1.4 Excess Carrier Distribution

**目标：求解中性区内过剩载流子浓度随距离的变化。**

**控制方程：稳态扩散方程（n区空穴为例）**

$$\frac{d^2(\delta p_n)}{dx^2} - \frac{\delta p_n}{L_p^2} = 0 \tag{8.9}$$

**物理含义：** 扩散（第一项）和复合（第二项）达到动态平衡。注入的空穴一边向n区深处扩散，一边与电子复合，浓度逐渐降低。

**通解：**

$$\delta p_n(x) = A e^{x/L_p} + B e^{-x/L_p} \quad (x \geq x_n) \tag{8.12}$$

**"长二极管"近似（$W_n \gg L_p$）：**

远离耗尽层处（$x \to \infty$），过剩载流子浓度必须趋于零 → $A = 0$。

利用边界条件 $\delta p_n(x_n) = p_{n0}[\exp(eV_a/kT) - 1]$：

$$\boxed{\delta p_n(x) = p_{n0} \left[\exp\left(\frac{eV_a}{kT}\right) - 1\right] \exp\left(-\frac{x - x_n}{L_p}\right)} \quad (x \geq x_n) \tag{8.14}$$

**物理图像：**
- 耗尽层边缘 $x = x_n$ 处浓度最大：$\delta p_n(x_n) = p_{n0}[\exp(eV_a/kT) - 1]$
- 随深入n区，以特征长度 $L_p$ 指数衰减
- 经过约 $3L_p$ 距离，浓度衰减到边缘值的约 5%

**$L_p$ 的物理意义：** 少数载流子在被复合前平均能扩散多远。

**p区同理（过剩电子）：**

$$\delta n_p(x) = n_{p0} \left[\exp\left(\frac{eV_a}{kT}\right) - 1\right] \exp\left(\frac{x + x_p}{L_n}\right) \quad (x \leq -x_p)$$

---

### 8.1.5 Derivation of the Ideal pn Junction Current

**从载流子分布到电流：**

**n区空穴扩散电流：**

$$J_p = -e D_p \frac{d(\delta p_n)}{dx}$$

对(8.14)求导，在耗尽层边缘 $x = x_n$：

$$J_p(x_n) = \frac{e D_p p_{n0}}{L_p} \left[\exp\left(\frac{eV_a}{kT}\right) - 1\right]$$

**p区电子扩散电流：**

$$J_n(-x_p) = \frac{e D_n n_{p0}}{L_n} \left[\exp\left(\frac{eV_a}{kT}\right) - 1\right]$$

**总电流（假设4c：耗尽层内电流恒定）：**

$$J = J_p(x_n) + J_n(-x_p)$$

定义**反向饱和电流密度**：

$$\boxed{J_s = \frac{e D_p p_{n0}}{L_p} + \frac{e D_n n_{p0}}{L_n}} \tag{8.26}$$

**理想二极管方程（肖克利二极管方程）：**

$$\boxed{J = J_s \left[\exp\left(\frac{eV_a}{kT}\right) - 1\right]} \tag{8.27}$$

**各项物理意义：**

| 项 | 含义 |
|----|------|
| $J_s$ | 反向饱和电流密度，$J_s \propto n_i^2$（极强温度依赖） |
| $\frac{e D_p p_{n0}}{L_p}$ | n区空穴电流贡献 |
| $\frac{e D_n n_{p0}}{L_n}$ | p区电子电流贡献 |
| $\exp(eV_a/kT)$ | 正向偏压降低势垒 → 注入载流子指数增加 |
| $-1$ | 保证零偏压时电流为零；反向偏压时 $J \to -J_s$ |

**$J_s$ 的另一种写法：**

$$\frac{e D_p p_{n0}}{L_p} = e \cdot \frac{L_p}{\tau_{p0}} \cdot p_{n0} = e \times \text{扩散速度} \times \text{平衡少数载流子浓度}$$

**电流分布特征：**
- n区中性区：空穴扩散电流指数衰减（复合），电子扩散电流相应增大
- 耗尽层内：电子电流和空穴电流各自为常数
- 总电流处处相等（假设4a）
- 对于 $p^+n$ 结（$N_a \gg N_d$），空穴电流远大于电子电流

### 8.1.6 Discussion and Examples

**正向电流的两个分量：**

耗尽层边缘哪种载流子电流占主导取决于掺杂浓度。对于 $p^+n$ 结（$N_a \gg N_d$）：
- $p_{n0} = n_i^2 / N_d$ 很大 → n区空穴浓度高
- $n_{p0} = n_i^2 / N_a$ 很小 → p区电子浓度低
- 所以空穴电流远大于电子电流

**中性区的电场：**

推导时假设中性区电场为零（只有扩散电流）。但实际上有很小的电场（约 1.5 V/cm 量级），比耗尽层内电场（约 $10^4$ V/cm）小几个数量级，可以忽略。

这个电场的来源：为了让总电流保持恒定，多数载流子需提供漂移电流来补充少数载流子扩散电流的减少。多数载流子浓度很高，很小的电场足以驱动所需电流。

**对数坐标下的I-V曲线：**

$\ln J$ vs $V_a$ 为直线，斜率 $= 1/V_t \approx 38.6 \text{ V}^{-1}$（室温硅）。

**工程经验法则：** 每增加约 60 mV 的正向偏压，电流增大 10 倍（$\ln(10) \times V_t \approx 0.060$ V）。

**典型数值：** 室温硅 $J_s \sim 10^{-11}$ A/cm²，正向 0.6 V 时电流约 mA 量级。

### 8.1.7 Temperature Effects

**温度影响的两处来源：**

$$J = J_s \left[\exp\left(\frac{eV_a}{kT}\right) - 1\right]$$

**1. $J_s$ 的温度依赖：**

$J_s \propto n_i^2 \propto T^3 \exp(-E_g / kT)$，极强的温度依赖。

硅pn结：温度每升高 10°C，$J_s$ 约增大 **4 倍**。

**2. 正向偏压下的温度效应：**

温度升高时：
- $J_s$ 增大 → 电流增大
- $\exp(eV_a/kT)$ 中 $T$ 在分母 → 同一 $V_a$ 下指数项减小 → 电流减小
- $J_s$ 的指数依赖占主导 → **同一偏压下，温度升高 → 电流增大**

反过来，保持电流恒定时，温度升高 → 所需正向偏压**减小**。

**Example 8.5 结论：**

硅pn结在 300K 时偏压 0.60V，温度升至 310K 时，保持同样电流需要偏压降至 **0.5827 V**，变化 **-17.3 mV**。

**工程经验法则（硅二极管）：**
- 反向饱和电流：温度每升高 10°C，约增大 4 倍
- 正向压降：温度每升高 1°C，约降低 2 mV（恒定电流下）
- 这个负温度系数可能导致**热失控**（功率二极管中需特别注意）

**GaAs vs Si：** GaAs 的 $E_g$ 更大（1.42 eV vs 1.12 eV），温度效应相对较弱（每 10°C 偏压降低约 12.3 mV）。

### 8.1.8 The "Short" Diode

**长二极管的假设：** p区和n区宽度远大于少数载流子扩散长度（$W_n \gg L_p$，$W_p \gg L_n$）。

**短二极管：** 一侧宽度远小于扩散长度（$W_n \ll L_p$）。

**边界条件：**
1. 耗尽层边缘 $x = x_n$：$p_n(x_n) = p_{n0} \exp(eV_a/kT)$（同前）
2. 欧姆接触处 $x = x_n + W_n$：$p_n(x_n + W_n) = p_{n0}$（过剩载流子为零，极高表面复合速度）

**通解（保留两项）：**

$$\delta p_n(x) = p_{n0} \left[\exp\left(\frac{eV_a}{kT}\right) - 1\right] \frac{\sinh\left[(x_n + W_n - x)/L_p\right]}{\sinh(W_n/L_p)} \tag{8.31}$$

**短极限 $W_n \ll L_p$（$\sinh\theta \approx \theta$）：**

$$\boxed{\delta p_n(x) = p_{n0} \left[\exp\left(\frac{eV_a}{kT}\right) - 1\right] \left(\frac{x_n + W_n - x}{W_n}\right)} \tag{8.33}$$

**过剩载流子浓度变为线性分布！**

**空穴扩散电流密度：**

$$\boxed{J_p = \frac{e D_p p_{n0}}{W_n} \left[\exp\left(\frac{eV_a}{kT}\right) - 1\right]} \tag{8.34}$$

**对比：**

| | 长二极管 | 短二极管 |
|---|---------|---------|
| 浓度分布 | 指数衰减 | 线性 |
| 电流 | $J_p = \frac{e D_p p_{n0}}{L_p}[\cdots]$ | $J_p = \frac{e D_p p_{n0}}{W_n}[\cdots]$ |
| 大小 | 较小 | 较大（因为 $W_n \ll L_p$） |
| 复合 | 显著 | 可忽略（电流为常数） |

**物理图像：** 长二极管中少数载流子有很长的"跑道"扩散和复合，浓度梯度较小。短二极管中载流子"跑不了多远"就撞到欧姆接触被复合，浓度梯度很大 → 更大的扩散电流。

**工程应用：**
- **太阳能电池**：短二极管结构让光生载流子在复合前被收集
- **双极型晶体管**：基区做得很短，减少复合、提高电流增益

## 8.2 Generation–Recombination Currents and High-Injection Levels

8.1节基于四条严格假设推导了理想二极管方程。8.2节逐一放松这些假设，分析实际二极管的非理想效应：
- **8.2.1**：耗尽层内的产生电流（反向偏压）和复合电流（正向偏压），违反假设4c
- **8.2.2**：高注入情况，违反假设3（低注入）

### 8.2.1 Generation–Recombination Currents

#### 8.2.1.1 反向偏压产生电流（Reverse-Biased Generation Current）

**物理图景：**

反向偏压下，耗尽层内电场很强，电子和空穴几乎被完全扫出——$n \approx 0$，$p \approx 0$。但半导体晶体中的深能级陷阱（由杂质或缺陷引入）会持续向导带发射电子、向价带发射空穴。在载流子被清扫殆尽的耗尽层内，这种发射过程反而加剧——陷阱持续产生新的电子-空穴对，它们立即被强电场分别扫向n区和p区，形成反向电流。

**本质：载流子不是从中性区扩散过来的，而是在耗尽层内部"凭空"产生出来的。**

**数学推导——SRH复合率：**

$$R = \frac{C_n C_p N_t (np - n_i^2)}{C_n(n + n') + C_p(p + p')}$$

耗尽层内 $n \approx p \approx 0$：

$$R = \frac{-C_n C_p N_t n_i^2}{C_n n' + C_p p'} \tag{8.36}$$

负号 → 不是复合，而是**产生**（generation）。

**简化：陷阱能级在本征费米能级处**（$n' = p' = n_i$）：

$$R = \frac{-n_i}{\tau_{p0} + \tau_{n0}}$$

定义平均寿命 $\tau_0 = (\tau_{n0} + \tau_{p0})/2$：

$$R = \frac{-n_i}{2\tau_0} \equiv -G \tag{8.40}$$

$G$ 为耗尽层内的产生率。

**产生电流密度：**

$$J_{gen} = \int_0^W eG \, dx = \frac{e n_i W}{2\tau_0} \tag{8.42}$$

**总反向电流密度：**

$$J_R = J_s + J_{gen} \tag{8.43}$$

$J_s$ 与电压无关，但 $J_{gen} \propto W$，而 $W \propto \sqrt{V_{bi} + V_R}$，所以**实际反向电流随反向电压增大而缓慢增大**。

**Example 8.6 结论（关键数据）：**

典型硅pn结在5V反偏下：
- $J_s \approx 4.16 \times 10^{-11}$ A/cm²
- $J_{gen} \approx 2.91 \times 10^{-7}$ A/cm²
- $J_{gen} / J_s \approx 7 \times 10^3$

**产生电流比理想反向饱和电流大了近4个数量级！** 在硅二极管中，产生电流是反向电流的主要成分。

**对比GaAs：** $n_i$ 极小，$J_s \propto n_i^2$ 和 $J_{gen} \propto n_i$ 都极小，但比值 $J_{gen}/J_s \propto 1/n_i$ 更大（$\sim 10^7$）。

#### 8.2.1.2 正向偏压复合电流（Forward-Bias Recombination Current）

**物理图景——复合电流是什么？**

理想模型假设（假设4c）：载流子从一边进入耗尽层，从另一边穿出，中间没有任何损失。

实际情况：半导体晶体中存在杂质原子和晶体缺陷，在禁带中引入深能级陷阱。载流子（电子和空穴）在穿越耗尽层时，可能被这些陷阱俘获而**直接复合**，没能到达中性区。

为了在中性区建立由边界条件决定的少数载流子浓度分布，必须从另一侧**额外注入更多载流子**来补偿这些在耗尽层内"损失"掉的部分。这些额外注入载流子形成的电流就是**复合电流**。

**理想模型的"理想"之处：**

回顾假设4c：耗尽层内电子电流和空穴电流各自为常数。这意味着理想模型假设载流子从一边进入耗尽层，从另一边穿出，**中间没有任何损失**。所有注入的电子都到达p区中性区，所有注入的空穴都到达n区中性区。

**实际的复合过程——陷阱的角色：**

实际半导体晶体不是完美无瑕的。晶体中不可避免地存在：
- **杂质原子**（非故意掺入的金属杂质如金、铁、铜等）
- **晶体缺陷**（位错、空位、间隙原子等）

这些杂质和缺陷会在禁带中引入**深能级**（deep energy levels），即陷阱能级（trap levels），位于禁带中央附近。陷阱在复合中扮演**"媒人"**角色——它先俘获一个电子，再俘获一个空穴，让它们复合。没有陷阱时，电子和空穴直接复合的概率很低（需同时满足能量守恒和动量守恒）。

这就是**SRH（Shockley-Read-Hall）复合机制**。

**复合电流的完整物理链条：**

1. **注入**：正向偏压下，p区空穴向n区注入，n区电子向p区注入
2. **穿越耗尽层**：载流子进入耗尽层，向对面运动
3. **陷阱俘获**：深能级陷阱俘获电子和空穴，一部分电子-空穴对在耗尽层内直接复合——载流子"牺牲"在耗尽层中
4. **浓度分布被破坏**：到达中性区的载流子减少了，少数载流子浓度低于边界条件预期的值，浓度梯度变小
5. **补偿机制**：为了维持由外加电压决定的少数载流子浓度分布（边界条件要求），必须从另一侧**额外注入更多载流子**来弥补损失
6. **形成额外电流**：这些额外注入的载流子在中性区流动，形成了**复合电流**

**为什么耗尽层内复合特别严重？**

关键在于耗尽层中心的**电子和空穴浓度同时较大**。

正向偏压下，用**准费米能级**描述载流子浓度：

$$n = n_i \exp\left[\frac{E_{Fn} - E_{Fi}}{kT}\right], \quad p = n_i \exp\left[\frac{E_{Fi} - E_{Fp}}{kT}\right]$$

在冶金结中心（$x = 0$），两个准费米能级关于本征费米能级对称分布：

$$E_{Fn} - E_{Fi} = E_{Fi} - E_{Fp} = \frac{eV_a}{2}$$

因此：

$$n = p = n_i \exp\left(\frac{eV_a}{2kT}\right)$$

**复合率 $R \propto np - n_i^2$**。在 $n = p$ 且两者都很大的情况下，复合率达到尖锐峰值（图8.14）。

**对比中性区：** 中性区多数载流子浓度极高，少数载流子浓度很低，复合率受限于少数载流子浓度，不高。

**数学推导：**

SRH复合率（$n' = p' = n_i$，$\tau_{n0} = \tau_{p0} = \tau_0$）：

$$R_{max} = \frac{n_i}{2\tau_0}\frac{[\exp(eV_a/kT) - 1]}{[\exp(eV_a/2kT) + 1]} \tag{8.51}$$

大偏压近似（$V_a \gg kT/e$）：

$$R_{max} = \frac{n_i}{2\tau_0}\exp\left(\frac{eV_a}{2kT}\right) \tag{8.52}$$

对耗尽层积分：

$$\boxed{J_{rec} = \frac{eWn_i}{2\tau_0}\exp\left(\frac{eV_a}{2kT}\right) = J_{r0}\exp\left(\frac{eV_a}{2kT}\right)} \tag{8.55}$$

**指数因子 $eV_a/2kT$ 的物理来源：** 耗尽层中心 $n = p \propto \exp(eV_a/2kT)$，只有 $V_a$ 的一半用于提高电子浓度，另一半用于提高空穴浓度。

**对比扩散电流：**

| 电流类型 | 指数因子 | 物理原因 |
|---------|---------|---------|
| 扩散电流 | $\exp(eV_a/kT)$ | 势垒降低 → 翻越势垒的载流子数 $\propto \exp(eV_a/kT)$ |
| 复合电流 | $\exp(eV_a/2kT)$ | 耗尽层中心 $n = p = n_i\exp(eV_a/2kT)$，复合率 $\propto np \propto \exp(eV_a/kT)$，但复合"有效区域"的载流子浓度本身 $\propto \exp(eV_a/2kT)$ |

**直观理解：** 扩散电流取决于"有多少载流子能翻越势垒"，这个数量对 $V_a$ 的依赖是 $\exp(eV_a/kT)$。复合电流取决于"耗尽层内电子和空穴同时存在且浓度都很高的区域"，而在耗尽层中心，$n$ 和 $p$ 各自只分到 $eV_a/2$ 的能量，所以增长只有扩散电流的一半快。

**工程意义：**

通过测量实际二极管的理想因子 $n$，可以判断哪种机制主导：
- $n \approx 2$：复合电流主导（低偏压）或高注入（高偏压）
- $n \approx 1$：扩散电流主导（正常工作区）
- $1 < n < 2$：两种机制共存

在高质量器件中，我们希望复合电流尽可能小。方法是使用高纯度材料（减少杂质陷阱）、减少晶体缺陷、避免引入深能级杂质。

**例外——发光二极管（LED）：** 在LED中复合是**需要的**（电子-空穴复合释放光子）。但LED使用直接带隙半导体（如GaAs）实现**辐射复合**，SRH复合（通过陷阱）是**非辐射复合**，以热的形式耗散能量，是效率损失。

#### 8.2.1.3 总正向电流与理想因子

**总正向电流密度 = 扩散电流 + 复合电流：**

$$J = J_D + J_{rec} \tag{8.56}$$

$$J_D = J_s \exp\left(\frac{eV_a}{kT}\right) \quad \text{（斜率 } \propto 1/V_t \text{，每60mV变化10倍）}$$

$$J_{rec} = J_{r0}\exp\left(\frac{eV_a}{2kT}\right) \quad \text{（斜率 } \propto 1/2V_t \text{，每120mV变化10倍）}$$

**关键参数对比：**
- $J_s$ 极小（硅 $\sim 10^{-11}$ A/cm²）
- $J_{r0}$ 远大于 $J_s$

**对数坐标下的斜率分析：**

$$\ln J_D = \ln J_s + \frac{V_a}{V_t} \quad \Rightarrow \quad \text{斜率} = \frac{1}{V_t} \approx 38.6 \text{ V}^{-1}$$

$$\ln J_{rec} = \ln J_{r0} + \frac{V_a}{2V_t} \quad \Rightarrow \quad \text{斜率} = \frac{1}{2V_t} \approx 19.3 \text{ V}^{-1}$$

**主导机制转换（图8.16）：**

| 偏压范围 | 主导电流 | 斜率 | 原因 |
|---------|---------|------|------|
| 低偏压 | **复合电流** $J_{rec}$ | $1/2V_t$ | $J_{r0} \gg J_s$，虽然指数增长慢，但系数大 |
| 高偏压 | **扩散电流** $J_D$ | $1/V_t$ | 指数增长更快，压倒 $J_{r0}$ 的系数优势 |

**理想因子（Ideality Factor）：**

$$I = I_s\left[\exp\left(\frac{eV_a}{nkT}\right) - 1\right] \tag{8.59}$$

| 条件 | 理想因子 $n$ | 主导机制 |
|------|-------------|---------|
| 低正向偏压 | $n \approx 2$ | 耗尽层复合主导 |
| 中等偏压 | $1 < n < 2$ | 过渡区域 |
| 高正向偏压 | $n \approx 1$ | 扩散电流主导 |

**$n$ 是拟合参数**，用于描述实际二极管与理想方程的偏差。通过测量 $n$ 可判断哪种机制主导。

### 8.2.2 High-Level Injection

**低注入假设的回顾：**

理想I-V推导中假设 $\delta p_n \ll p_{p0}$，注入的少数载流子浓度远小于多数载流子浓度，多数载流子浓度基本不变。

**高注入的发生：**

正向偏压继续增大，$\delta n$ 和 $\delta p$ 不断增加，最终可能与多数载流子浓度可比拟甚至超过：

$$\delta n \gtrsim n_{no}, \quad \delta p \gtrsim p_{po}$$

**数学处理：**

质量作用定律：

$$(n_o + \delta n)(p_o + \delta p) = n_i^2 \exp\left(\frac{V_a}{V_t}\right)$$

高注入下 $\delta n \approx \delta p \gg n_o, p_o$：

$$\delta n \cdot \delta p \approx n_i^2 \exp\left(\frac{V_a}{V_t}\right)$$

由 $\delta n = \delta p$（电中性）：

$$\boxed{\delta n = \delta p \approx n_i \exp\left(\frac{V_a}{2V_t}\right)} \tag{8.62}$$

**对比：**

| | 低注入 | 高注入 |
|---|--------|--------|
| 过剩载流子浓度 | $\delta n = n_{p0}[\exp(V_a/V_t)-1] \propto \exp(V_a/V_t)$ | $\delta n \approx n_i \exp(V_a/2V_t)$ |
| 电流 | $I \propto \exp(V_a/V_t)$ | $I \propto \exp(V_a/2V_t)$ |
| 理想因子 | $n = 1$ | $n = 2$ |

**物理含义：** 高注入时，注入的载流子开始改变整个中性区的载流子浓度分布，需要同时增加电子和空穴的浓度，需要更多的能量。

**图8.17展示的完整I-V曲线（三个区域）：**

1. **低偏压**：复合电流主导，$n \approx 2$
2. **中等偏压**：扩散电流主导，$n \approx 1$
3. **高偏压**：高注入，$n \approx 2$（再次）

**有趣的对称性：** 耗尽层复合和高注入都导致 $n = 2$，但物理机制完全不同——前者是陷阱辅助的耗尽层内复合，后者是载流子浓度改变了中性区分布。

**高注入的另一个效应：基区电导调制**——大量载流子涌入中性区，显著提高中性区电导率，在大功率器件中非常重要。

## 8.2 总结

| 非理想效应 | 发生条件 | 表现 | 理想因子 $n$ |
|-----------|---------|------|-------------|
| 耗尽层复合电流 | 低正向偏压 | $J \propto \exp(eV_a/2kT)$ | $n \approx 2$ |
| 耗尽层产生电流 | 反向偏压 | $J_R = J_s + J_{gen}$，$J_{gen} \propto W(V_R)$ | — |
| 高注入 | 高正向偏压 | $J \propto \exp(eV_a/2kT)$ | $n \approx 2$ |

**典型硅二极管中的主导地位：**
- **反向**：产生电流远大于 $J_s$，是主要成分
- **低正向**（$< 0.4$ V）：复合电流主导
- **正常正向**（0.6~0.7 V）：扩散电流主导，接近理想
- **高正向**（$> 0.8$ V）：高注入使电流增长变缓

**抑制复合电流的方法：** 使用高纯度材料（减少杂质陷阱）、减少晶体缺陷、避免引入深能级杂质。

**例外——发光二极管（LED）：** LED中复合是**需要的**（释放光子），但使用直接带隙半导体（如GaAs）实现辐射复合，SRH复合（通过陷阱）是非辐射复合，是能量损失。

## 8.3 Small-Signal Model of the pn Junction

前面8.1和8.2节讨论的都是**直流（dc）特性**。但在实际电路中，二极管经常工作在有交流信号叠加在直流偏置上的场景（整流纹波、线性放大、调制解调等）。小信号模型的核心思想是：**在直流工作点附近，把非线性的二极管I-V特性线性化**，用线性电路元件（电阻、电容）来等效。

### 8.3.1 Diffusion Resistance（扩散电阻）

**从非线性到线性：**

二极管I-V特性是指数型（非线性）的：

$$I_D = I_s\left[\exp\left(\frac{eV_a}{kT}\right) - 1\right] \tag{8.64}$$

在直流偏压 $V_0$ 处叠加很小的交流信号 $\hat{v}_1 \sin(\omega t)$（$\hat{v}_1 \ll V_t$），可在 $V_0$ 处做泰勒展开保留线性项：

$$I_D(V_0 + \hat{v}_1) \approx I_D(V_0) + \left.\frac{dI_D}{dV_a}\right|_{V_0} \cdot \hat{v}_1$$

**增量电导与扩散电阻：**

增量电导定义为直流I-V曲线在工作点处的斜率：

$$g_d = \left.\frac{dI_D}{dV_a}\right|_{V_a = V_0} \tag{8.65}$$

扩散电阻（增量电阻）为其倒数：

$$r_d = \left.\frac{dV_a}{dI_D}\right|_{I_D = I_{DQ}} \tag{8.66}$$

**物理含义：** $r_d$ 是二极管对交流信号的等效电阻，等于直流I-V曲线在对应电流处的斜率。

**具体计算：**

正向偏压足够大时忽略 $-1$ 项：

$$g_d = \left(\frac{e}{kT}\right) I_s \exp\left(\frac{eV_0}{kT}\right) = \frac{I_{DQ}}{V_t} \tag{8.67}$$

$$\boxed{r_d = \frac{V_t}{I_{DQ}}} \tag{8.68}$$

**扩散电阻的特性：**

| 特性 | 说明 |
|------|------|
| 与偏置电流成反比 | 电流越大，I-V曲线越陡，$r_d$ 越小 |
| 与热电压成正比 | $V_t = kT/e$，温度越高 $r_d$ 越大 |
| 典型值 | 1 mA 偏置下，$r_d \approx 26 \Omega$（室温硅） |

**直观理解：** 把I-V曲线想象为山坡。低电流时山坡平缓（斜率小 → $r_d$ 大），高电流时山坡陡峭（斜率大 → $r_d$ 小）。

---

### 8.3.2 Small-Signal Admittance（小信号导纳）

#### 扩散电容的物理起源

正向偏压下，n区存储大量过剩空穴，p区存储大量过剩电子。偏压越大 → 注入越多 → 存储电荷越多。当偏压变化时，存储电荷随之变化 → 电荷的充放电产生额外的交流电流。

**这就是扩散电容的物理本质：少数载流子电荷存储效应导致的电容。**

**两种电容的对比：**

| | 结电容 $C_j$ | 扩散电容 $C_d$ |
|---|-------------|---------------|
| 来源 | 耗尽层内离子电荷随电压变化 | 中性区少数载流子存储电荷随电压变化 |
| 偏压依赖 | 反偏时显著，正偏时较小 | 正偏时显著，反偏时可忽略 |
| 数量级 | $\sim$pF | $\sim$nF（比结电容大3~4个数量级） |
| 物理位置 | 耗尽层 | 中性区 |

#### 定性理解（图8.19）

1. 直流偏压 $V_{dc}$ 下，n区空穴浓度呈指数衰减分布
2. 叠加交流信号后，耗尽层边缘浓度在 $p_{dc}[1 \pm \hat{v}_1/V_t]$ 之间波动
3. 交流正半周浓度曲线上移，负半周下移
4. **阴影面积 $\Delta Q$** 就是交流周期内充放电的电荷量

**类比：** 活塞泵。直流偏压决定平衡位置（平均存储电荷），交流信号让活塞来回运动，每次循环吸入/排出一定量"流体"（电荷）。

#### 数学推导

**第一步：边界条件**

总电压 $V_a = V_0 + v_1(t)$，耗尽层边缘：

$$p_n(0, t) = p_{dc} \exp\left[\frac{e v_1(t)}{kT}\right]$$

小信号近似（$|v_1| \ll V_t$）：

$$p_n(0, t) \approx p_{dc}\left[1 + \frac{\hat{V}_1}{V_t}e^{j\omega t}\right] \tag{8.74}$$

**第二步：求解时间相关扩散方程**

n区过剩空穴满足：

$$D_p \frac{\partial^2(\delta p_n)}{\partial x^2} - \frac{\delta p_n}{\tau_{p0}} = \frac{\partial(\delta p_n)}{\partial t} \tag{8.75}$$

设解为直流分量 + 交流分量：$\delta p_n(x, t) = \delta p_0(x) + p_1(x) e^{j\omega t}$

代入后分离时间相关项，得到交流分量方程：

$$\frac{d^2 p_1(x)}{dx^2} - \frac{(1 + j\omega\tau_{p0})}{L_p^2} p_1(x) = 0 \tag{8.80}$$

**注意 $(1 + j\omega\tau_{p0})$ 的出现**——时间导数项 $j\omega p_1$ 的贡献，使等效"复扩散长度"变为 $L_p/\sqrt{1+j\omega\tau_{p0}}$。

**第三步：求解交流分量**

边界条件：$p_1(\infty) = 0$，$p_1(0) = p_{dc}(\hat{V}_1/V_t)$

$$p_1(x) = p_{dc}\left(\frac{\hat{V}_1}{V_t}\right) e^{-C_p x} \tag{8.84}$$

其中 $C_p^2 = (1 + j\omega\tau_{p0})/L_p^2$。

**第四步：计算交流扩散电流**

$$j_p(t) = -eD_p \frac{\partial p_1}{\partial x}\bigg|_{x=0} e^{j\omega t} = eD_p C_p p_{dc}\left(\frac{\hat{V}_1}{V_t}\right) e^{j\omega t}$$

总交流电流相量（$A$ 为结面积）：

$$\hat{I}_p = I_{p0}\sqrt{1+j\omega\tau_{p0}}\left(\frac{\hat{V}_1}{V_t}\right) \tag{8.95}$$

其中 $I_{p0} = eAD_p p_{dc}/L_p$ 为直流空穴电流分量。

同理，p区电子贡献：

$$\hat{I}_n = I_{n0}\sqrt{1+j\omega\tau_{n0}}\left(\frac{\hat{V}_1}{V_t}\right) \tag{8.96}$$

**第五步：总导纳**

$$Y = \frac{\hat{I}_p + \hat{I}_n}{\hat{V}_1} = \frac{1}{V_t}\left[I_{p0}\sqrt{1+j\omega\tau_{p0}} + I_{n0}\sqrt{1+j\omega\tau_{n0}}\right] \tag{8.98}$$

#### 低频近似

当 $\omega\tau_{p0} \ll 1$ 且 $\omega\tau_{n0} \ll 1$ 时：

$$\sqrt{1+j\omega\tau} \approx 1 + \frac{j\omega\tau}{2} \tag{8.100}$$

代入导纳表达式，分离实部和虚部：

$$\boxed{Y = g_d + j\omega C_d} \tag{8.103}$$

**扩散电导：**

$$\boxed{g_d = \frac{I_{p0} + I_{n0}}{V_t} = \frac{I_{DQ}}{V_t}} \tag{8.104}$$

与8.3.1节结果完全一致。

**扩散电容：**

$$\boxed{C_d = \frac{I_{p0}\tau_{p0} + I_{n0}\tau_{n0}}{2V_t}} \tag{8.105}$$

#### 扩散电容的物理意义

观察公式 $C_d = \frac{1}{2V_t}(I_{p0}\tau_{p0} + I_{n0}\tau_{n0})$：

- $I_{p0}\tau_{p0}$ 代表**n区存储的空穴电荷总量**（电流 × 寿命 = 电荷）
- $I_{n0}\tau_{n0}$ 代表**p区存储的电子电荷总量**
- 除以 $2V_t$ 反映了电荷随电压的变化率

**更深刻的理解：** $C_d \approx g_d \cdot \tau_{eff}$，即电导与载流子寿命的乘积。**载流子寿命越长 → 存储电荷越多 → 充放电效应越显著 → 电容越大。**

#### Example 8.7 的关键结论

硅pn结在 $I_{DQ} = 1$ mA，$\tau_{p0} = 10^{-7}$ s 下（假设 $N_a \gg N_d$，空穴电流主导）：

$$C_d = \frac{I_{DQ}\tau_{p0}}{2V_t} = \frac{10^{-3} \times 10^{-7}}{2 \times 0.0259} = 1.93 \text{ nF}$$

$$r_d = \frac{V_t}{I_{DQ}} = \frac{0.0259}{10^{-3}} = 25.9 \text{ }\Omega$$

**扩散电容（约2 nF）比反偏结电容（约几个 pF）大了3~4个数量级！** 在正向偏压下，扩散电容是主要的电容成分。

**扩散电容随偏置电流的变化：** $C_d \propto I_{DQ}$，偏置电流越大 → 存储电荷越多 → 扩散电容越大。这与结电容的行为相反（结电容在反偏时更大）。

### 8.3.3 Equivalent Circuit（等效电路）

**完整的小信号等效电路（图8.22b）：**

```
        rs          rd     Cd
  ───┤├────┤├────┤├────┤├────
      串联    扩散   扩散
      电阻    电阻   电容
              (并联)
        │             │
        └─── Cj ──────┘
            结电容
```

**各元件的含义：**

| 元件 | 名称 | 来源 | 公式 |
|------|------|------|------|
| $r_d$ | 扩散电阻 | I-V曲线斜率 | $r_d = V_t / I_{DQ}$ |
| $C_d$ | 扩散电容 | 少数载流子电荷存储 | $C_d = \frac{I_{p0}\tau_{p0} + I_{n0}\tau_{n0}}{2V_t}$ |
| $C_j$ | 结电容 | 耗尽层离子电荷 | $C_j = \frac{\epsilon_s A}{W}$（第7章） |
| $r_s$ | 串联电阻 | 中性区欧姆电阻 | 取决于几何尺寸和掺杂 |

**各元件的相对大小：**

- **正向偏压下**：$C_d \gg C_j$，结电容常被忽略
- **$r_d$**：mA级电流下约几十欧姆
- **$r_s$**：通常很小（几个到几十欧姆），大多数情况下可忽略
- **反向偏压下**：$C_d \approx 0$，$C_j$ 主导，$r_d$ 极大

**串联电阻的影响：**

实际二极管的端电压为：

$$V_{app} = V_a + I \cdot r_s \tag{8.106}$$

其中 $V_a$ 是结电压（理想I-V方程中的电压），$V_{app}$ 是外加总电压。串联电阻使实际I-V曲线向右偏移（图8.23）：要达到相同电流，需要更高的外加电压。

大多数二极管中 $r_s$ 可忽略，但在某些器件中（如双极型晶体管中），串联电阻可能处于反馈回路中，被增益放大后变得不可忽略。

## 8.3 总结

**核心概念：**

| 概念 | 定义 | 物理起源 |
|------|------|---------|
| 扩散电阻 $r_d$ | 直流I-V曲线斜率 | 指数I-V特性的局部线性化 |
| 扩散电容 $C_d$ | 少数载流子存储电荷的充放电 | 偏压变化 → 注入量变化 → 存储电荷变化 |
| 结电容 $C_j$ | 耗尽层离子电荷的偏压依赖 | 耗尽层宽度随偏压变化 |

**关键公式：**

$$r_d = \frac{V_t}{I_{DQ}}, \quad C_d = \frac{I_{DQ}\tau_{eff}}{2V_t}, \quad Y = g_d + j\omega C_d$$

**等效电路（正向偏压）：**

```
  ─── rs ───┬── rd ───┬──
            │         │
            Cd   (Cj)  │
            │  通常    │
            │  可忽略  │
            └─────────┘
```

**设计启示：**
- **高频应用**：需减小扩散电容 → 减小载流子寿命（掺金）或减小偏置电流
- **低噪声放大器**：需低 $r_s$ → 优化欧姆接触和中性区导电性
- **快速开关**：需快速释放存储电荷 → 减小 $\tau$（寿命缩短技术）

# 05 局域优化

## 5.1 维度灾难（Curse of Dimensionality）

在许多分子动力学的问题中，我们希望找出某个稳定的结果，或是某个相对好的结果。这类问题属于优化问题，我们需要考虑许多优化算法。有人可能会问：既然使用计算机计算，为什么不采用穷举的方法？这样不是能直接列出最优的情况吗？

理查德·贝尔曼提出维度灾难的概念：当参数空间维度 $d$ 增加时，空间体积指数增长 $V \propto L^d$，导致：

- 穷举搜索计算量爆炸    
- 梯度信息稀疏化
- 局部极小值数量急剧增加    

**解决思路**：采用**局域优化**方法，从初始点出发，沿局部梯度信息迭代搜索。

求解最优解的过程，就是求解某一个函数极值的过程。

## 5.2 最速下降（梯度下降）

### 5.2.1 最小二乘法

首先回顾接触过的优化问题——最小二乘法。

以拟合数组 $c = a x + b y$ 为例，我们可以计算一个拟合结果与实际结果的误差：
$$
z = \sum_{i=1}^{n} (c_i - a_i x - b_i y)^2
$$

这个误差存在着一个极小值，使得这个误差达到极小值的参数就是我们要的拟合参数，因此有 **极值条件**：
$$
\frac{\partial z}{\partial x} = 0, \quad \frac{\partial z}{\partial y} = 0
$$

数据拟合问题本质上是**高维函数优化问题** ，目标函数 $z(x,y)$ 的极小值点即为最优拟合参数。这是概率学中的最概然估计告诉我们的。

在分子动力学的模拟中，我们就是需要通过找到某个函数的极值来获得最优解，一般是极小值，这就用到最速下降法/梯度下降法。

### 5.2.2 梯度下降

假定我们有一个目标函数 $f(x)$ ，想要求其最小值，显然，沿着 **梯度的负方向** 来移动就能轻易到达，这是一个很直观的想法。在选定某个初始坐标后，我们就可以通过沿着负梯度的方向不断移动坐标：
$$
\vec{x}_{k+1} = \vec{x}_k - \alpha_k \nabla f(\vec{x}_k)
$$
其中， $\alpha_{k}$ 为每一次更新坐标所使用的步长。当 $x_{k}$ 接近某个极小值时，函数的梯度 $\nabla f(x)$ 就会无限接近于0，这个时候更新的步长就会极小，不断 **逼近极小值** 。理论上永远无法到达极小值，因此我们在更新的时候需要设置一个判断标准 $\varepsilon$ ，当两次的函数值相差极小或两次的坐标相差极小时结束计算。即当 $f(x_{k+1}) - f(x_{k}) < \varepsilon$ 或 $x_{k+1} - x_{k} < \varepsilon$ 时停止。

**两个核心要素**：
1. **搜索方向**：负梯度方向 $-\nabla f(\vec{x}_k)$，函数下降最快方向
2. **步长选择**： $\alpha_k$ 一般来说很小，这是为了防止更新的步长太大反而发散，保证 $f(\vec{x}_{k+1}) < f(\vec{x}_k)$

此外，对于一般问题/简单问题而言，梯度下降中的步长 $\alpha$ 是一个固定值，但对于某些复杂的问题，可以使用特殊的算法如 SGD/Momentum/Adam 等算法来实时改变 $\alpha$ 的值，使得能够 **更快地达到最优解，也使得有概率跳出局部最优** 。

### 5.2.3 代码详解

```matlab
 clear
 x = 3;                    % 初始点x₀=3
 gx = 4*x^3 - 3;           % 计算梯度f'(x) = 4x³ - 3
 i = 1;                    % 迭代计数器
 y = x^4 - 3*x - 10;       % 初始函数值f(x₀)
 
 while abs(gx) > 0.001     % 收敛条件：梯度模长小于阈值
     alpha = 2;            % 初始步长设为2（可尝试较大值）
     tp = []; ys = [];     % 存储步长与对应函数值
   tp(1) = 0; ys = y;    % 记录步长0时的状态（当前点）
   
   nx = x - gx*alpha;    % 试探新位置：x_new = x - α·∇f(x)
   ny = nx^4 - 3*nx - 10;% 计算新位置的函数值
   
   % 二分搜索：确保函数值下降
   while ny > y          % 若f(x_new) > f(x)，步长过大
       alpha = alpha/2;  % 步长减半
       nx = x - gx*alpha;% 重新计算新位置
       ny = nx^4 - 3*nx - 10;
       tp = [tp; alpha]; % 记录步长
       ys = [ys; ny];    % 记录对应函数值
   end
   
   % 插值准备：选取初始点附近的3个点
   tp = [tp(1); tp(end-1:end)]; % 取α=0和最后两个试探点
   ys = [ys(1); ys(end-1:end)]; % 对应函数值
   
   as = linspace(min(tp), max(tp), 100); % 在步长区间精细采样
   yspline = spline(tp, ys, as);         % 三次样条插值
   [q, qq] = min(yspline);               % 找插值函数最小值
   
   nx = x - gx*as(qq);   % 用最优插值步长更新位置
   x = nx;               % 更新当前点
   d(i) = x;             % 记录优化轨迹
   y = x^4 - 3*x - 10;   % 更新函数值
   gx = 4*x^3 - 3;       % 更新梯度
   i = i + 1;            % 迭代计数+1
end
```

#### 1. 初始化（第2-5行）

- **初始点选择**：`x = 3` 远离极值点，便于观察完整优化过程
- **目标函数**：$f(x)=x^4−3x−10$ ，其导数为 $g(x)=4x^3−3$
- **收敛阈值**：0.001，控制精度与效率的平衡

#### 2. 步长搜索策略（第8-22行）

这是算法最精妙的部分：

**为什么初始步长设为2？**

- 经验性选择，需满足**足够大以快速接近极值** + **可接受范围内测试**
- 若太小（如0.01），需要更多次迭代；若太大（如100），可能跳过最优区域

**二分搜索机制**

在每一次更新坐标之后，都判断新坐标的目标函数是否更小，如果不更小则减小步长并重新计算坐标

```matlab
while ny > y          % 若f(x_new) > f(x)，步长过大
   alpha = alpha/2;  % 步长减半
   nx = x - gx*alpha;% 重新计算新位置
   ny = nx^4 - 3*nx - 10;
   tp = [tp; alpha]; % 记录步长
   ys = [ys; ny];    % 记录对应函数值
end
```

- **保证下降**：通过不断折半步长，确保最终满足 $f(x_{new​})<f(x)$
- **理论保证**：根据连续函数介值定理，必存在使函数下降的步长区间

**记录搜索轨迹**： `tp`和`ys`数组记录所有试探的(步长, 函数值)对，为后续插值提供数据支撑

#### 3. 插值优化（第25-31行）

二分法找到的步长仅是"可接受"的，未必"最优"，通过插值能够直接找到接近最优的步长。然后再使用这个接近最优的步长来更新坐标。

#### 4. 更新与迭代（第32-38行）

- **位置更新**：使用插值得到的最优步长`as(qq)`
- **轨迹记录**：`d(i)`数组保存所有迭代点，便于事后分析收敛路径
- **梯度更新**：新梯度用于下一轮方向计算

## 5.3  二维最速下降法

### 5.3.1 分析

对于二维函数，其实核心也不变——沿着负梯度的方向更新坐标，直到抵达最小值。与一维的区别仅仅在于：

- 计算的梯度也变成二维向量
- 需要对每一个维度的坐标用相应梯度单独更新

举一个简单的粒子，有一个 **目标函数**：$f(\vec{x}) = \dfrac{x_1^2}{a} + \dfrac{x_2^2}{b}, \quad a=2, b=3$ 。想要求其最小值，我们只需要三步：

1. 计算梯度：$\nabla f = [\dfrac{2x_1}{a}, \dfrac{2x_2}{b}]$
2. 从 $\vec{x}_1 = [a, b] = [2, 3]$ 出发，$\nabla f(\vec{x}_1) = [2, 2]$ ，每一个位置独立计算梯度
3. 迭代坐标 $\vec{x}_{k+1} = [\dfrac{a(a-b)^k}{(a+b)^k}, \dfrac{b(a-b)^k}{(a+b)^k}]$

> 上述迭代坐标是在已知初始位置是直接计算的确切表达式，实际上较少使用这种方式，具体见 [[#5.2.2 坐标更新]]

**收敛特性**：

- 当 $a = b$ 时，一次迭代即达极小值点
- 当 $a \neq b$ 时，收敛速度较慢，在极值点附近出现**锯齿现象**（zigzagging）

**算法实现**：

1. 随机初始化 $\vec{x}_0$ 在圆周上
2. 计算梯度和当前函数值
3. 采用与一维类似的**二分+插值**策略确定步长
4. 沿负梯度方向更新位置
5. 重复迭代20次

**可视化特征**：优化路径呈现折线状，验证了收敛速度慢的特点。

**拓展补充**：最速下降法的锯齿现象源于相邻搜索方向正交，这是其固有缺陷。改进方法包括**共轭梯度法**（Conjugate Gradient）和 **BFGS** 等拟牛顿法。

### 5.3.2 代码及其结果

```matlab
clear 
a = 2; b = 3;            % 定义椭圆函数参数
x = linspace(-1, 1);     % 绘图用网格
y = linspace(-1, 1);
z = [];
for i = 1:length(x)
    for j = 1:length(y)
        z(i,j) = x(i)^2/a + y(j)^2/b; % 计算目标函数值
    end
end
contour(x, y, z, 30);    % 绘制30条等高线
hold on;

% 随机初始化起点
t = rand * 2*pi;
x = [2*cos(t); 2*sin(t)]; % 起点在半径为2的圆上随机
ally = [];

% 主优化循环
for iter = 1:20
    y = x(1)^2/a + x(2)^2/b;              % 当前函数值
    gx = [2*x(1)/a; 2*x(2)/b];            % 梯度∇f = [2x₁/a, 2x₂/b]
    
    alpha = 10;                           % 初始步长10
    nx = x - gx * alpha;                  % 试探新位置
    ny = nx(1)^2/a + nx(2)^2/b;
    
    tp(1) = 0; ys = y;                    % 记录初始状态
    
    % 二分搜索
    while ny > y
        alpha = alpha/2;
        nx = x - gx * alpha;
        ny = nx(1)^2/a + nx(2)^2/b;
        tp = [tp; alpha];
        ys = [ys; ny];
    end
    
    % 插值优化（同于一维）
    tp = [tp(1); tp(end-1:end)];
    ys = [ys(1); ys(end-1:end)];
    as = linspace(min(tp), max(tp), 100);
    yspline = spline(tp, ys, as);
    [q, qq] = min(yspline);
    
    x = x - gx * as(qq);                  % 更新位置向量
    d(iter, :) = x;                       % 记录轨迹
    ally = [ally; q];                     % 记录函数值历史
    
    % 绘制实时轨迹
    plot(d(:,1), d(:,2), 'r*-');         % 红色星号连线
    drawnow;                              % 立即刷新图像
end
```

### 5.3.3 实际坐标更新

事实上，在实际应用中的二维梯度下降并非直接计算每一步 $x_{k}$ 的确切表达式，仍然是通过梯度来进行更新:
$$
\begin{array}{c}
f= f(x, y) \\
g(x, y) = \nabla f = \left[\dfrac{\partial f}{\partial x}, \dfrac{\partial f}{\partial y} \right] = [g_{1}, g_{2}] \\
\left\{ \begin{array}{l}
x_{k + 1} = x_{k} - \alpha_{x} \cdot g_{1}  \\
y_{k + 1} = y_{k} - \alpha_{y} \cdot g_{2}
\end{array} \right.
\end{array}
$$
分别计算 $x, y$ 的梯度并分别更新，才是最常见的做法。

### 5.3.4 拓展与延伸

#### 1. 算法局限性

1. **锯齿现象**：梯度方向正交导致收敛慢，可通过**动量法**（Momentum）或 **共轭梯度法** 改善
2. **病态问题**：对条件数大的函数（如狭长山谷）效率低下
3. **局部最优**：无法保证全局最优性

## 5.4 模拟退火算法

### 5.4.1 算法的思想

模拟退火（Simulated Annealing, SA）算法由Kirkpatrick等人于1983年提出，其灵感源于**金属热处理**中的退火工艺：

1. **高温加热**：原子获得高动能，**脱离局部晶格位置**，可以自由移动
2. **缓慢冷却**：原子逐步失去动能，有足够时间重新排列，最终形成能量最低的稳定晶格结构（全局最优）

**核心洞察**：热激发使系统能够**越过能量势垒**，逃离局部极小值。

在分子动力学中，**温度**是控制体系探索能力的关键参数：

- **高温（T=50）**：体系接近气体，粒子自由运动，可跨越势垒
- **低温（T=1）**：体系接近固体，粒子被限制在势阱中

模拟退火正是利用这一特性，**主动调控温度**，引导体系 **从任意初始状态向全局能量最低点演化**。

### 5.4.2 数学模型

#### 1. Metropolis 准则

模拟退火的灵魂是**概率性接受准则**，基于统计力学中的玻尔兹曼分布：

$$
P(\Delta E) = \begin{cases}
1, & \Delta E \leq 0 \quad (\text{能量降低，必然接受}) \\
\exp\left(-\dfrac{\Delta E}{k_B T}\right), & \Delta E > 0 \quad (\text{能量升高，概率接受})
\end{cases}
$$

其中：

- $\Delta E = E_{new} - E_{old}$：能量变化
- $k_B$：玻尔兹曼常数（通常设为1）
- $T$：当前温度

**物理意义**：在温度 T 下，**系统接受能量变高的状态的概率服从玻尔兹曼分布**。温度越高，接受概率越大。这样做是为了在高温下能够跳出当前的势阱。


## 四、多粒子体系局域优化

### 4.1 惰性气体体系优化模型

**势能函数**：采用Lennard-Jones势
$$
U(r) = 4\epsilon \left[ \left(\frac{\sigma}{r}\right)^{12} - \left(\frac{\sigma}{r}\right)^6 \right]
$$
其中 $\sigma$ 为粒子直径，$\epsilon$ 为势阱深度。代码中采用简化形式 $U(r) = r^{-12} - 2r^{-6}$。

**力计算**：
$$
F = -\nabla U = -\left(-\frac{12}{r^{13}} + \frac{12}{r^7}\right) \frac{\vec{r}}{r}
$$

**算法特点**：
- 去掉速度项，仅根据力控制步长
- 步长自适应：$\alpha = 0.01 / \sqrt{\max(f_x^2 + f_y^2)}$
- 每步势能单调下降，保证收敛性

**初始结构影响**：
1. **正方晶格+随机扰动**：最终形成**三角晶格**（二维密堆积）
2. **Eden模型随机结构**：形成类似三角晶格的"碎片"

**拓展补充**：三角晶格是二维体系能量最低结构，配位数为6，对应面心立方（FCC）的(111)面。

### 4.2 模拟退火算法（Simulated Annealing）

**物理灵感**：模仿金属退火过程——高温使原子脱离局部亚稳态，缓慢冷却实现全局能量最低。

**算法实现**：
```matlab
T0 = 5 * abs(sin(i/600*pi));
```
- **温度波形**：正弦振荡 + 绝对值，非负且周期性变化
- **振幅**：5（温度波动幅度）
- **周期**：600步

**关键作用**：周期性升温提供**逃离局部极小**的动能，降温过程逐步稳定到更低能量状态。

**拓展补充**：更规范的模拟退火应遵循**Metropolis准则**：
$$
P = \min\left(1, \exp\left(-\frac{\Delta E}{k_B T}\right)\right)
$$
接受使能量增大的移动，接受概率随温度降低而减小。

### 4.3 Basin-Hopping算法

**核心思想**：**局部优化 + 随机扰动** 的迭代循环
1. **跳跃（Hop）**：对当前结构施加随机扰动/大尺度移动
2. **局部优化**：使用最速下降法或共轭梯度法快速弛豫到邻近局域极小值（盆地）
3. **接受判断**：基于Monte Carlo准则决定接受新结构

**优势**：通过跳跃机制探索不同势能盆地（basin），避免陷入单一局部极小。

**文献应用**：
- 研究对象：三维惰性气体团簇
- 成果：系统收录于**剑桥团簇数据库**（Cambridge Cluster Database）
- 网站：`http://www-wales.ch.cam.ac.uk/CCD.html`

**拓展补充**：Basin-Hopping是目前寻找**原子团簇全局最优结构**最有效算法之一，成功应用于Lennard-Jones、Morse等各类势函数体系。

## 五、关键知识点总结与拓展

### 5.1 核心算法对比

| 方法 | 核心思想 | 优点 | 缺点 | 适用场景 |
|------|----------|------|------|----------|
| **最速下降法** | 负梯度方向 | 简单，稳定 | 收敛慢，锯齿现象 | 低维、初始阶段 |
| **模拟退火** | 热激发搜索 | 可逃离局部极小 | 参数敏感，计算量大 | 组合优化、结构搜索 |
| **Basin-Hopping** | 跳跃+局部优化 | 效率高，找到全局最优 | 实现复杂 | 团簇结构预测 |

### 5.2 步长选择策略总结

1. **固定步长**：简单但可能振荡或收敛慢
2. **二分搜索**：保证下降，稳健但效率低
3. **插值优化**：二分后精细搜索，平衡效率与精度
4. **自适应步长**：根据最大受力调整，适合多粒子体系

**理论准则**：Wolfe条件（充分下降+曲率条件）是线搜索的数学标准。

### 5.3 维度灾难的应对策略

- **降维**：主成分分析（PCA）、特征选择
- **随机采样**：蒙特卡洛方法
- **启发式搜索**：遗传算法、粒子群优化
- **机器学习**：用神经网络拟合势能面

### 5.4 实际应用案例

1. **材料科学**：晶体结构预测（USPEX、CALYPSO软件）
2. **药物设计**：分子构象搜索
3. **生物物理**：蛋白质折叠
4. **催化化学**：催化剂活性位点优化

5. **逃离局部极小**：需要热激发（模拟退火）或结构跳跃（basin-hopping）
6. **结构表征工具**：关联函数连接微观结构与宏观物性

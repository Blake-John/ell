在 MATLAB 中，可以使用 `num2str` 函数或者 `sprintf` 函数将向量格式化输出成字符串。

  1. **使用 `num2str` 函数**

     * `num2str` 函数可以将数值数组转换为字符串。对于向量，它会将向量中的元素依次转换为字符串，并用空格分隔。
     * 例如：

       ```matlab
       vec = [1 2 3 4 5];
       str = num2str(vec);
       disp(str);
       ```
       输出结果为：`1 2 3 4 5`。这里 `vec` 是一个向量，通过 `num2str` 函数将其转换为字符串 `str`，元素之间以空格分隔。

     * 如果想要指定小数位数等格式，可以使用带有格式参数的 `num2str` 函数。例如：

       ```matlab
       vec = [1.2345 2.3456 3.4567];
       str = num2str(vec, '%.2f');
       disp(str);
       ```
       输出结果为：`1.23 2.35 3.46`。这里通过格式说明符 `%.2f` 指定保留两位小数。

  2. **使用 `sprintf` 函数**

     * `sprintf` 函数是一个更强大的格式化输出函数。它可以按照指定的格式将向量中的元素输出为字符串。
     * 例如：

       ```matlab
       vec = [10 20 30 40 50];
       str = sprintf('%d %d %d %d %d', vec);
       disp(str);
       ```
       输出结果为：`10 20 30 40 50`。这里通过 `%d` 格式说明符来输出整数，每个 `%d` 对应向量中的一个元素。

     * 对于小数向量，可以使用 `%f` 格式说明符，并且可以指定精度。例如：

       ```matlab
       vec = [1.2345 2.3456 3.4567];
       str = sprintf('%.2f, %.2f, %.2f', vec);
       disp(str);
       ```
       输出结果为：`1.23, 2.35, 3.46`。这里通过 `%.2f` 指定保留两位小数，并且用逗号分隔元素。

这两种方法都可以根据需要对向量进行格式化输出，`num2str` 函数比较简单直接，而 `sprintf` 函数则提供了更多灵活的格式控制选项。

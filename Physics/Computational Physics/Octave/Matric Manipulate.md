# 01 逐元素操作

在 Octave 中，可以使用 **点运算符（.）** 来进行 **矩阵逐元素操作** 。

> [!tip]
> 点运算符 `.` 其实是来源于数学中的 **点乘** ，点乘将对应向量/矩阵的元素逐一相乘，因此这里使用 `.` 来标示对向量/矩阵的逐元素操作

# 1. 矩阵每个元素取倒数

要对矩阵的每个元素取倒数，可以使用 `1 ./ A`，其中 `A` 是矩阵。这里的 `./` 是逐元素除法运算符，`1` 表示一个与 `A` 维度相同的矩阵，其每个元素都是 1。

**示例代码：**

```octave
A = [1, 2; 3, 4];  % 定义一个 2x2 矩阵
inv_A = 1 ./ A;    % 对每个元素取倒数
disp(inv_A);       % 显示结果
```

**输出结果：**

```
1.00000  0.50000
0.33333  0.25000
```

# 2. 两个矩阵的对应元素相除

要将两个矩阵的对应元素相除，可以使用 `A ./ B`，其中 `A` 和 `B` 是两个维度相同的矩阵。这里的 `./` 运算符会对两个矩阵的对应元素进行逐元素除法。

**示例代码：**

```octave
A = [1, 2; 3, 4];  % 定义第一个矩阵
B = [5, 6; 7, 8];  % 定义第二个矩阵
result = A ./ B;   % 对应元素相除
disp(result);      % 显示结果
```

**输出结果：**

```
0.20000  0.33333
0.42857  0.50000
```

## 总结

- 矩阵每个元素取倒数：`1 ./ A`
- 两个矩阵对应元素相除：`A ./ B`

这两种操作都使用了 Octave 中的逐元素运算符 `./`，确保了对矩阵的每个元素进行操作而不是矩阵的整体运算。

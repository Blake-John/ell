# 1.4.1 Discrete-time unit step & unit impulse（离散单位阶跃与单位脉冲）  

## 1. 单位脉冲信号  #def/信号与系统/Signals 
- **定义**：  
	$$  
	\delta[n] =  
	\begin{cases}  
	1, & n = 0 \\  
	0, & n \neq 0  
	\end{cases}  
	$$
	
	![[imgs/discrete_unit_impluse.png|200]]  
- **延时形式**：  
	- $$\delta[n-2] = \begin{cases} 1, & n = 2 \\ 0, & n \neq 2 \end{cases}$$
	  ![[imgs/discrete_unit_impluse_2.png|200]]
    - $$\delta[n^2 - 4] = \begin{cases} 1, & n^2 - 4 = 0 \\ 0, & n^2 - 4 \neq 0 \end{cases}$$
      ![[imgs/discrete_unit_impluse_3.png|200]]
  
- **与函数相乘**：  
	- $$x[n]\delta[n] = x[0]\delta[n] = \begin{cases} x[0], & n = 0 \\ 0, & n \neq 0 \end{cases}$$
	- $$x[n]\delta[n - k] = x[k]\delta[n - k] = \begin{cases} x[k], & n = k \\ 0, & n \neq k \end{cases}$$
	  ![[imgs/discrete_unit_impluse_multiply_with_func.png|500]]  

## 2. 单位阶跃信号  
- **定义**：  
	$$  
	u[n] =  
	\begin{cases}  
	1, & n \geq 0 \\  
	0, & n < 0  
	\end{cases}  
	$$
	![[imgs/discrete_unit_step.png|200]]  
- **延时形式**：  
	$$  
	u[n - 2] = \begin{cases} 1, & n \geq 2 \\ 0, & n < 0 \end{cases}  
	$$
	![[imgs/discrete_unit_step_2.png|200]]  
- **与函数相乘**：  
  $$  
  x[n]u[n] = \begin{cases} x[n], & n \geq 0 \\ 0, & n < 0 \end{cases}  
  $$  

## 3. $u[n]$ 与 $\delta[n]$ 的关系  

1. $$\delta[n] = u[n] - u[n - 1]$$  
2. $$u[n] - u[n - 2] = \delta(n) + \delta(n - 1)$$  
3. $$u[n] = \sum_{m = -\infty}^{n} \delta[m] = \sum_{k = 0}^{\infty} \delta[n - k]$$（如何证明？）  

# 1.4.2 Continuous-time unit step & unit impulse（连续单位阶跃与单位冲激）  

## 1. 单位阶跃  
- **定义**：  
	$$  
	u(t) =  
	\begin{cases}  
	1, & t > 0 \\  
	0, & t < 0  
	\end{cases}  
	$$
	  ![[imgs/conti_unit_step.png|200]]
- **延时形式**：  
	$$  
	u(t - t_0) = \begin{cases} 1, & t > t_0 \\ 0, & t < t_0 \end{cases}  
	$$
	![[imgs/conti_unit_step_2.png|200]]  
- **组合形式**：  
	$$  
	G(t) = u(t) - u(t - t_0)  
	$$
	![[imgs/compose_unit_step.png|200]]
	- $G(t)$ 可用于表示分段函数，例如：  
		$$  
		F(t) =  
		\begin{cases}  
		f(t), & 0 < t < t_0 \\  
		0, & \text{else}  
		\end{cases}  
		= f(t)[u(t) - u(t - t_0)]  
		$$
		![[imgs/compose_func.png|200]]  

## 2. 单位冲激函数 - Dirac Delta Function  
- **定义**：  
	$$  
	\delta(t) =  
	\begin{cases}  
	\infty, & t = 0 \\  
	0, & t \neq 0  
	\end{cases}  \tag{1.4.1}
	$$
	![[imgs/dirac_delta_func.png|200]]  
  且应满足：  
  $$  
  \begin{cases}  
  \int_{-\infty}^{\infty} \delta(t) dt = 1 \\  
  u(t) = \int_{-\infty}^{t} \delta(\tau) d\tau  
  \end{cases} \quad \Rightarrow \quad \delta(t) = \dfrac{du(t)}{dt}  \tag{1.4.2}
  $$
  > [!note] 
  > 实际上， $\delta(t)$ 的定义是由 式 1.4.2 得到的，$\delta(t)$  应满足上式而导出的 式 1.4.1
  
- **延时形式**：  
	$$  
	\delta(t - t_0) = \begin{cases} \infty, & t = t_0 \\ 0, & t \neq t_0 \end{cases}  
	$$
	  ![[imgs/conti_unit_impluse_2.png|200]]
- **与函数相乘**：  
	- $$x(t)\delta(t) = \begin{cases} x(0), & t = 0 \\ 0, & t \neq 0 \end{cases} = x(0)\delta(t) \neq x(0)$$  
	- $$x(t)\delta(t - t_0) = \begin{cases} x(t_0), & t = t_0 \\ 0, & t \neq t_0 \end{cases} = x(t_0)\delta(t - t_0) \neq x(t_0)$$  
	- **注意**：结果 **仍为一个函数**，是被抽样的函数，而 **非一个常数**。  
- **示例**：  
  若 $x(t) = 2u(t - 1) - 3u(t - 2) + 2u(t - 4)$，则 $$x'(t) = 2\delta(t - 1) - 3\delta(t - 2) + 2\delta(t - 4)$$

# 1.6.1 System with & without memory
## 1. 无记忆系统 - 即时系统、瞬时系统
- 例：
	- $V(t) = R i(t) \rightarrow y(t) = R x(t)$，包含 $y(t) = \sin t$，$y[n] = x^2[n]$，$y[n] = (2x[n] - x^2[n])^2$。
	- **恒等系统** **Identity System** #def/信号与系统/Signals ：
		- $y(t) = x(t)$，$y[n] = x[n]$。

### 2. 记忆系统 - 动态系统
- 例：
	- 累积
		- $V(t) = \dfrac{1}{C} \int_{-\infty}^{t} i(\tau) d\tau \rightarrow y(t) = \dfrac{1}{C} \int_{-\infty}^{t} x(\tau) d\tau$
		- $y[n] = \sum_{k=-\infty}^{n} x[k] \Leftrightarrow y[n] = y[n-1] + x[n]$
	- 时移
		- $y(t) = x(t-1)$

# 1.6.2 Invertibility & Inverse Systems（可逆性与逆系统）

类比函数可逆性，定义域与值域 **一一对应** 时存在逆函数。

输入输出一一对应：

$$x[n] \rightarrow \text{system 1} \rightarrow \text{system 2} \rightarrow x[n]$$

当两个系统级联成恒等系统时，两系统 **可逆** 且互为 **逆系统** #def/信号与系统/Signals 。

- 例如，以下系统均不可逆
	- $y[n] = x[n]x[n-1]$
	- $y(t) = x^2(t)$
	- $y(t) = \sin(t)$

# 1.6.3 Causality 因果性

- **因果系统** #def/信号与系统/Signals ：输出 **只与当前及之前输入有关** ，与 **以后的输入无关**
- **非因果系统** #def/信号与系统/Signals ：输出与 **之后输入有关**。

例如
  - 因果：
	  - $y(t) = \int_{-\infty}^{t} x(\tau) d\tau$
	  - $y[n] = \sum_{k=-\infty}^{n} x[k]$
  - 非因果：
	  - $y(t) = x(-t)$
	  - $y(t) = x(t+1)$
  - 特殊：
	  - $y(t) = x(t)\cos(t+1)$（因果，输入仅 $x(t)$
	  - $y[n] = x[n] - x[n+1]$（非因果）

> [!question] 如何判断系统是否是因果的？
> 对于一个系统而言，输入只有 $x(t)$ 或 $x[n]$ ，**其余的参量均与因果性无关**

# 1.6.4 稳定性 Stability

- **输入有界、输出有界** 则 **稳定** #def/信号与系统/Signals （$y(t) < B$，$x(t) < A$ for any $t$）。
- 例：
	- 稳定：
		- $y(t) = e^{x(t)}$
	- 不稳定：
		- $y(t) = t x(t)$
		- $y(t) = \int_{-\infty}^{t} x(\tau) d\tau$
		- $y[n] = \sum_{k=-\infty}^{n} x[k]$

# 1.6.5 时不变系统 Time Invariance

如果输入有 **一个时间上的平移**，**输出也产生相同的平移**，除此之外无任何变化，则系统是 **时不变** #def/信号与系统/Signals 的，否则是 **时变** #def/信号与系统/Signals 的。 $x(t) \rightarrow y(t) \Rightarrow x(t-t_0) \rightarrow y(t-t_0)$（时不变）。

## 检验方法：
  
  1. 建立系统模型。
  2. 令 $x_1(t) = x(t-t_0)$，确定 $y_1(t)$。
  3. 检验 $y_1(t)$ 与 $y(t-t_0)$ 是否相等。

## 例：

对于 $y(t) = \sin[x(t)]$ ，当 $x_{1}(t) = x(t - t_{0})$ 时， $y_{1}(t) = \sin[x(t - t_{0})]$ ，与 $y(t - t_{0}) = \sin[x(t - t_{0})]$ 保持一致，故时不变。

对于 $y(t) = x(2t)$ ，当 $x_{1}(t) = x(t - t_{0})$ 时， $y_{1}(t) = x(2t - t_{0})$ ，与 $y(t - t_{0}) = x(2(t - t_{0}))$ 不一致，故时变。

# 1.6.6 线性 Linearity

- **齐次性 (Homogeneity)**：$x_1(t) \rightarrow y_1(t) \Rightarrow a x_1(t) \rightarrow a y_1(t)$。
- **叠加性 (Superposition)**：$x_2(t) \rightarrow y_2(t) \Rightarrow a x_1(t) + b x_2(t) \rightarrow a y_1(t) + b y_2(t)$。

既满足齐次性，又满足叠加性的系统称为 **⌈线性系统⌋** #def/信号与系统/Signals 。

例：
- 线性：$y(t) = t x(t)$ ，当 $x(t) \rightarrow a x(t)$ 时，$y(t) \rightarrow a y(t)$
- 非线性：$y(t) = x(t) + b$ ，当$x(t) \rightarrow a x(t)$ 时，$y(t) = ax(t) + b \neq a y(t) = ax(t) + ab$ 

> [!attention] 
> 线性系统可用线性方程表示，但线性方程 **未必** 对应线性系统。

# 1.6.7 线性增量系统 Incrementally Linear System

例：$y(t) = k x(t) + 3$，$x_1(t) = a x(t) \rightarrow y_1(t) = a k x(t) + 3 \neq a y(t)$。

**任何一个增量线性系统可等效为线性系统 + 与输入无关的响应** ，即 线性增量系统 = 线性系统 + 增量 = 线性系统 + 零输入响应

若零输入响应为 0，系统线性，也称系统为 **松弛的** 或 **静止的**。

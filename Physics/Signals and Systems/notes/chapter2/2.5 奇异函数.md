# 2.5.1 单位脉冲作为理想化短脉冲

> [!important] 
> $\delta(t)$ 是许多短脉冲的理想化信号。

## 1. 类似于 $\delta(t)$ 的其他信号

由于 $\delta(t) = \delta(t) * \delta(t)$，且 $\lim_{\Delta \to 0} \delta_\Delta(t) = \delta(t)$。

记 $r_\Delta(t) = \delta_\Delta(t) * \delta_\Delta(t)$ 

![[imgs/r_delta_t.png]]

则有 $\left.r_{\Delta}(t)\right|_{t = \Delta} = \int_{0}^{\Delta} \left( \dfrac{1}{\Delta} \right)^{2}d\tau = \dfrac{1}{\Delta}$

当 $\Delta \to 0$ 时，$r_\Delta(t) \to \delta(t)$，即

$$
\lim_{\Delta \to 0} r_\Delta(t) = \delta(t)
$$


## 2.  分析短脉冲信号 $\to \delta(t)$ 的条件

$\delta(t)$ 在实际应用中是一个 $\Delta$ 极小的 **短脉冲**，不同系统需要的 $\Delta$ 的精度不同。

# 2.5.2 通过卷积定义单位脉冲

我们关心的是 $\delta(t)$ **与其他信号作用会有什么效果**。

## 定义1

$$
x(t) = x(t) * \delta(t)
$$

当 $x(t) = 1$ 时，

$$
\int_{-\infty}^\infty \delta(\tau) x(t - \tau) d\tau = 1
$$

## 定义2

由于 $g(-t) = g(-t) * \delta(t) = \int_{-\infty}^\infty \delta(\tau) g(-(t - \tau)) d\tau = \int_{-\infty}^\infty g(\tau - t) \delta(\tau) d\tau$

当 $t = 0$ 时，

$$
g(0) = \int_{-\infty}^\infty g(\tau) \delta(\tau) d\tau
$$

如果一个信号 $g(t)$ 经过一个系统后输出为 $g(0)$，则该系统就是 $\delta(t)$。

$\langle f(t), \phi(t) \rangle = \int_{\Omega} f(t) \phi(t) dt = \phi(0)$，其中 $\Omega$ 为 $\phi (t)$ 的定义域， 此时 $f(t) = \delta(t)$，$\delta(t)$ 为广义函数。

# 2.5.3 单位脉冲的性质

## 由定义推出的性质

1. $C * \delta(t) = C$

2. $\int_{-\infty}^\infty x(t) \delta(t) dt = x(0) = \int_{-\infty}^\infty x(0) \delta(t) dt$
	- $x(t) \delta(t) = x(0) \delta(t)$
	- $x(t) \delta(t - t_0) = x(t_0) \delta(t - t_0)$

3. $x(t) = t \to t \delta(t) = 0$

4. $\delta(t)$ 为 **偶函数**
	- $\int_{-\infty}^\infty \delta(-t) x(t) dt \overset{\tau = -t}{=} \int_{+\infty}^{-\infty} \delta(\tau) x(-\tau) d(-\tau) = \int_{-\infty}^\infty \delta(\tau) x(0) d\tau = x(0) = \int_{-\infty}^{+\infty}x(t)\delta(t)dt$ 
	- 所以 $x(t) \delta(-t) = x(t) \delta(t)$，即 $\delta(-t) = \delta(t)$

5. $\int_{-\infty}^\infty x(t) \delta(at) dt \overset{\tau = at}{=} \frac{1}{a} \int_{-\infty}^\infty x\left(\frac{\tau}{a}\right) \delta(\tau) d\tau$ 
	- 当 $a > 0$ 时，$\int_{-\infty}^\infty x(t) \delta(at) dt = \frac{1}{a} x(0) = \frac{1}{a} \int_{-\infty}^\infty x(t) \delta(t) dt$
	- 当 $a < 0$ 时，$\int_{-\infty}^\infty x(t) \delta(at) dt = -\frac{1}{a} x(0) = -\frac{1}{a} \int_{-\infty}^\infty x(t) \delta(t) dt$
	- 可得：$x(t) \delta(at) = \frac{1}{|a|} x(t) \delta(t) \implies \delta(at) = \frac{1}{|a|} \delta(t)$

# 2.5.4 单位脉冲的微分 —— 单位冲激偶 (Unit Doubets)

## 1. 定义

设 $y(t) = \frac{d x(t)}{dt}$，则 $\delta(t) \to \frac{d x(t)}{dt} \to y(t) = \frac{d \delta(t)}{dt}$

记 $\frac{d \delta(t)}{dt} = u_1(t)$，称为 **⌈单位冲激偶⌋（unit doublet）** #def/信号与系统/Signals 

## 2. $\frac{d \delta(t)}{dt}$ 的波形

$$
\lim_{\Delta \to 0} \frac{d \delta_\Delta(t)}{dt} = \lim_{\Delta \to 0} \frac{1}{\Delta} (\delta(t) - \delta(t - \Delta)) = \frac{d \delta(t)}{dt}
$$

![[imgs/unit_doublet.png]]

## 3. 积分定义

从 $x(t) = x(t) * \delta(t)$，我们得到

$$
\frac{d}{dt} (x(t) * \delta(t)) = x'(t) * \delta(t) = x(t) * \delta'(t) = x'(t)
$$

则有

$$x(t)*\delta'(t) = x'(t)$$

考虑 $x(t) = 1$，

$$
0 = \int_{-\infty}^\infty x(t - \tau) \delta'(\tau) d\tau = \int_{-\infty}^\infty \delta(\tau) d\tau = \int_{-\infty}^\infty \delta'(\tau) d\tau
$$

所以 $\int_{-\infty}^\infty u_1(t) dt = 0$，即 **积分后上下相互抵消**，这个结果可以从图像中很轻易地看出。

## 4. 性质

1. $\delta'(t)$ 是奇函数，$\delta'(-t) = -\delta'(t)$
	- 偶函数的导数为奇函数

2. $\frac{d}{dt} [x(t) \delta(t)] = \frac{d}{dt} [x(0) \delta(t)] = x(0) \delta'(t)= x'(t) \delta(t) + x(t) \delta'(t) = x'(0) \delta(t) + x(t) \delta'(t)$
	- 因此：
		- $x(0) \delta(t) = x'(0) \delta(t) + x(t) \delta'(t)$
	    - $x(t) \delta'(t) = x(0) \delta'(t) - x'(0) \delta(t)$
3. $t \delta'(t) = t|_0 \delta'(t) - t'|_0 \delta(t) = 0 - \delta(t) = -\delta(t)$

## 5. $\delta(t)$ 的 $k$ 阶微分

定义 $h(t)$ 为一个对输入信号进行 **求导** 的系统，则该系统的响应为 $u_{1}(t) = \delta(t)*h(t)$ ，如下图所示：

![[imgs/u1.png]]

当我们输入一个普通信号 $x(t)$ 时，该系统的响应为 $y(t) = \dfrac{dx(t)}{dt} = x(t)*u_{1}(t)$ ，此时 $u_{1}(t)$ 与 $x(t)$ 作用的效果等于 $\delta'(t)$ ，则我们将 **单位冲激偶** 记为 $\delta'(t) \triangleq u_{1}(t)$ 。

由上可知，其二阶微分为 :

$$\delta''(t) = \delta(t)*u_{1}(t) \triangleq u_{2}(t)$$

那么，可以推出其 k 阶微分为 :

$$u_k(t) = \underbrace{u_1(t) * \dots * u_1(t)}_{k}$$

## 6. $\delta(t)$ 的积分

在 [[../chapter1/1.4 单位脉冲函数与单位阶跃函数#3. $u[n]$ 与 $ delta[n]$ 的关系|1.4]] 中，我们已经知道了单位脉冲信号和单位阶跃信号的关系，对于连续时间来说，他们之间的关系即为 : 

$$u(t) = \int_{-\infty}^t \delta(\tau) d\tau$$

因此，我们将其记为 $u(t) \triangleq u_{-1}(t)$

即 $u_{-1}(t)$ 为 $\delta(t)$ 的 **一次积分** 。 

> [!seealso] 
> $$x(t) * u(t) = \int_{-\infty}^\infty x(\tau) u(t - \tau) d\tau = \int_{-\infty}^t x(\tau) d\tau$$
> $h(t) \to u(t)$ 是一个 **积分器**

## 7. $\delta(t)$ 的高次积分

由于 $u(t)$ 本身是一个 **积分器** ，因此我们可以通过使用 $u(t)$ 来做卷积运算得到 $\delta(t)$ 的高次积分 :

$$\begin{align}
&u_{-2}(t) = u(t) * u(t) = \int_{-\infty}^t u(\tau) d\tau = tu(t) \\
&u_{-3}(t) = u_2(t) * u(t) = \int_{-\infty}^t u_{{-2}}(\tau) d\tau = \frac{1}{2} t^2 u(t) \\
&u_{-4}(t) = u_3(t) * u(t) = \int_{-\infty}^t u_{-3}(\tau) d\tau = \frac{1}{6} t^3 u(t)\\ &\vdots \\
&u_{-k}(t) = \underbrace{u(t) * \dots * u(t)}_{k} = \frac{t^{k-1}}{(k-1)!} u(t) \tag{2.5.1}
\end{align}
$$

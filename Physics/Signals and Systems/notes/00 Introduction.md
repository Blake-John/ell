# 1 信号相关概念

### 1.1 什么是信号

信号经过抽象、简化可以表示为函数。信号输入系统后产生输出响应，即系统对输入的信号进行处理。

信号的处理通常不在时域上考虑，而是在频域上考虑，思考一个信号是由多少不同频率的函数合成的。

考虑的方面包括：
1. 对信号进行分类
2. 对系统进行分析，对信号进行处理，使其达到我们想要的输出
3. 讨论信号在系统中如何变化

### 1.2 系统的功能

一个系统需要的功能：
1. 对信号进行采样，需保证不失真且成本低
2. 对采样的信号进行分析，在频域上分析
3. 对信号进行还原

### 1.3 主要任务 - 信号与系统分析

**基本目的：**
1. 揭示信号自身的特性，如时间、频率特性
2. 信号发生改变时，信号特性的相应改变

**基本思想：**
设法将信号分解成简单的基本单元信号的线性组合

**基本方法：**
在时域、频域、变换域对信号进行分解

**分析对象：**
线性时不变系统（Linear Time - Invariant System）

**基本任务：**
1. 求已知系统对给定输入信号所产生的响应
2. 根据已知激励与响应，求系统的特性

**具体内容：**
1. 信号分析 - 特征分析
2. 系统分析 - 如何响应输入
3. 系统设计 - 信号处理
4. 信号设计 - 满足特定信号要求

# 2.3.1 类氢光谱

里德伯常量的计算公式为：
$$R_H=\dfrac{2 \pi^2 m_e e^4}{(4 \pi \varepsilon_0)^2 h^3 c}$$
若将 $e^2 \rightarrow Z e^2$，即原子序数大于1时，得到：
$$R=\dfrac{2 \pi^2 m_e (Z e^2)}{(4 \pi \varepsilon_0)^2 h^3 c}$$
代入氢光谱公式：
$$\sigma=R\left(\dfrac{1}{m^2}-\dfrac{1}{n^2}\right)=R_H\left(\dfrac{1}{(m/Z)^2}-\dfrac{1}{(n/Z)^2}\right)$$
当 $m=2Z$ 时，有：
$$\sigma=R_H\left(\dfrac{1}{2^2}-\dfrac{1}{(n/Z)^2}\right),\quad n=3,4,5,\ldots$$
对于He原子，$Z=2$，则：
$$\sigma=R_H\left(\dfrac{1}{2^2}-\dfrac{1}{(n/2)^2}\right),\quad n=m+1,m+2,\ldots=5,6,7,\ldots$$
令 $k=\dfrac{n}{2}=\dfrac{5}{2},\dfrac{6}{2},\dfrac{7}{2},\ldots$，此时与里德伯指出的氢光谱的经验公式一致。

# 2.3.2 里德伯常量的变化

对于不同原子， **原子核质量的改变导致R值改变**。此外，电子与核之间为 **二体问题**，共同绕质心运动，因此需要进行对计算公式进行修正。

设 $m$ 和 $m_e$ 为核与电子的质量，$r_1$、$r_2$ 为两质心的距离，$r$ 为两者距离，则：
$$r_1 + r_2 = r,\quad m r_1 = m_e r_2$$
解得：
$$r_1 = \dfrac{m_e}{m + m_e} r,\quad r_2 = \dfrac{m}{m + m_e} r$$
两粒子受向心力为：
$$\dfrac{m v^2}{r_1} = \dfrac{m_e v_e^2}{r_2} = \dfrac{1}{4 \pi \varepsilon_0} \dfrac{Z e^2}{r^2}$$
又 $v = \omega r_1$，$v_e = \omega r_2$，则：
$$\dfrac{m m_e}{m + m_e} r \omega^2 = \dfrac{1}{4 \pi \varepsilon_0} \dfrac{Z e^2}{r^2} \tag{2.3.1}$$
令 $\mu = \dfrac{m m_e}{m + m_e}$ 为 **[[折合质量]]**。 ^9fada5

又 **角动量应是量子化的**，则：
$$m v r_1 + m_e v_e r_2 = n \dfrac{h}{2 \pi}$$
代入 $v$、$v_e$、$r_1$、$r_2$，可得：
$$\mu \omega r^2 = n \dfrac{h}{2 \pi}$$
用此式消去 $\omega$，并代入 [[2.3 类氢光谱#^9fada5|式 2.3.1]] ，有：
$$r = \dfrac{4 \pi \varepsilon_0 n^2 h^3}{4 \pi^2 \mu e^2 Z},\quad n=1,2,3,\ldots$$
同理计算体系的能量，有：
$$E = -\dfrac{2 \pi^2 \mu (Z e^2)^2}{(4 \pi \varepsilon_0)^2 h^2} \cdot \dfrac{1}{n^2}$$
计算 $R_A$，有：
$$R_A = \dfrac{2 \pi^2 \mu (Ze^2)^2}{(4 \pi \varepsilon_0)^2 h^3 c} = \dfrac{2 \pi m_e (Ze^2)^2}{(4 \pi \varepsilon_0)^2 h^3 c} \cdot \dfrac{1}{1 + \dfrac{m_e}{m}}$$

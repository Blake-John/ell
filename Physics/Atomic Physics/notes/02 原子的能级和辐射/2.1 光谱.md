# 2.1.1 问题的提出

核外电子是如何排布的，为什么会有如下的几种性质？

## 1. 原子的稳定性

### a. 电子的量子化分布

原子有电子（量子化的）出射，不应将 -ze 认为是均匀分布于 $10^{-10}\ m$ 球内，也 **不能设想电子在核周围做对称分布**（**静止的**），因这样无法使原子获力平衡。

### b. 电子绕核转动的电磁辐射问题

若假设电子绕核转动，带电粒子加速运动，**有电磁辐射**！

此时，有能量损耗（动能减少），则

$$
\dfrac{mv^2}{r} < \dfrac{e^2}{4\pi\varepsilon_0 r^2}
$$

为维持运动，则 **电子运动半径减小**（即库仑力除了一部分提供向心力外，剩余的力将电子拉向原子核）

![[imgs/原子落入原子核.png|200]]

电子最终 **以螺旋运动方式落入原子核**，则一切原子都不复存在，事实否定这一点！

## 2. 原子的同一性

Rutherford 模型实为经典的行星模型；为什么美国、俄罗斯、月球上铁和中国一样，用行星模型无法解释，按行星模型，因太阳系的形成与初始条件有关

## 3. 原子的再生性

为什么 **原子同外来粒子发生作用后，外来粒子远离，原子又恢复原状**？而天体中行星与别的彗星撞击，则行星原状态被打乱。

3. X 射线的产生

# 2.1.2 原子光谱和原子光谱一般情况

## 1. 光谱

**⌈光谱⌋** #def/原子物理/原子结构  是电子辐射 **波长成分**（定性）和 **强度**（定量）的记录（光谱实验分定性和定量分析）。

![[imgs/各金属的光谱.png]]

## 2. 测量仪器

其测量是由光谱仪（包括光源、分光器 - 棱镜或光栅、记录仪）来测量。

![[imgs/光谱仪结构.png]]

## 3. 分类

**a. 分立光谱**：**稀薄气体** 或 **金属的蒸汽**（此时原子出于游离状态）所发射的光谱，其特点是 **谱线分立**，每条线都对应不同波长。

> 分离光谱也称为 **线状光谱**

![[imgs/氢原子的光谱.png]]

**b. 连续光谱**：**炽热的固体**、**液体** 及 **高压气体所发射的光谱**，其特点是 **波长连续变化**（由一系列所有波长的光组成）。

![[imgs/连续光谱.png]]

**c. 带状光谱** ：有些光源的光谱是分段密集的

## 4. 发射与吸收

**a. 吸收光谱**：高温的固体发出白光（由连续分布的一切波长的光组成），**白光通过某物质**时，白光中**某些波长的光会被该物质吸收**，在原来白光的背底上留下一些被吸收（缺失）的波长所形成的线条。显然，该物质所**吸收的光恰好就是该物质高温（激发）时所能发出的光**.

![[imgs/吸收光谱.png]]

**b. 发射光谱**：光源发射出的光谱

## 5. 氢原子光谱

1885 年瑞士一位中学数学教师 J.Balmer 在对这些谱线分析研究后，将它们纳入一个经验的数学公式 — **J.Balmer 公式**：

$$
\lambda = B \dfrac{n^2}{n^2 - 4} \quad n = 3, 4, 5, \cdots
$$

我们令 $\sigma = \dfrac{1}{\lambda} = \dfrac{1}{B} \dfrac{n^2 - 4}{n^2} = \dfrac{4}{B} \left( \dfrac{1}{2^2} - \dfrac{1}{n^2} \right)$ ，则有 :

$$
\sigma = R_H \left( \dfrac{1}{2^2} - \dfrac{1}{n^2} \right) \quad n = 3, 4, 5, \ldots
$$

其中 $R_H = \dfrac{4}{B}$ 为 **氢的里德伯常量** #def/原子物理/原子结构 ，而 $\sigma$ 称为 **⌈波数⌋** #def/原子物理/原子结构 。且 $R_H = 1.0967758 \times 10^7 \, \text{m}^{-1}$

后来，氢原子的其它谱线系也被发现。

> 值得注意的是，目前里德伯常量为一个 **经验常量** ，没有数学公式的依据

## 六、各谱线系公式

| 谱线系名称                | 公式表达式                                                         | $m$ 的值 | $n$ 的取值范围             |
| :------------------- | :------------------------------------------------------------ | :----- | :-------------------- |
| **T. Lyman 赖曼系**     | $\sigma = R_H \left( \dfrac{1}{1^2} - \dfrac{1}{n^2} \right)$ | 1      | $n = 2, 3, 4, \ldots$ |
| **J.J. Balmer 巴耳末系** | $\sigma = R_H \left( \dfrac{1}{2^2} - \dfrac{1}{n^2} \right)$ | 2      | $n = 3, 4, 5, \ldots$ |
| **F. Paschen 帕邢系**   | $\sigma = R_H \left( \dfrac{1}{3^2} - \dfrac{1}{n^2} \right)$ | 3      | $n = 4, 5, 6, \ldots$ |
| **F. Brackett 布拉开系** | $\sigma = R_H \left( \dfrac{1}{4^2} - \dfrac{1}{n^2} \right)$ | 4      | $n = 5, 6, 7, \ldots$ |
| **H.A. Pfund 普丰德系**  | $\sigma = R_H \left( \dfrac{1}{5^2} - \dfrac{1}{n^2} \right)$ | 5      | $n = 6, 7, 8, \ldots$ |

综上，$\sigma = R_H \left( \dfrac{1}{m^2} - \dfrac{1}{n^2} \right) = T(m) - T(n)$

其中 T 称为 **⌈光谱项⌋** #def/原子物理/原子结构 

氢原子的光谱项为 $T(n) = \dfrac{R_H}{n^2}$，$n = 1, 2, 3, \ldots$ 

![[imgs/氢原子谱线系.png|300]]

## 7. 总结

1. 原子光谱是 **线状的**，**分立的**
2. 不同谱线由 **共同光谱项相联系**
3. 每一 **谱线波数** 为 **两光谱项之差**.

对每一个 $m$（$m = 1, 2, \ldots$）和 $n = m + 1, m + 2, \ldots$ 构成一谱线系.

$$
\sigma = T(m) - T(n) \quad m = 1, 2, 3, \ldots, \quad n = m + 1, m + 2, \ldots
$$

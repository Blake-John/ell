## Explaination

> [!PDF|important] [01 Vector Analysis, p.44](Physics/Electrodynamics/chapters/01%20Vector%20Analysis.pdf#page=44&selection=111,0,122,56&color=important)
> > The **source of the problem is the point r = 0**, where v blows up (and where, in Eq. 1.84, we have unwittingly divided by zero)
> 
> r = 0 处是问题的关键

> [!PDF|note] [01 Vector Analysis, p.44](Physics/Electrodynamics/chapters/01%20Vector%20Analysis.pdf#page=44&selection=122,58,142,29&color=note)
> > It is quite true that **∇ · v = 0 everywhere except the origin**, but right **at the origin the situation is more complicated**
> 
> 除了 r = 0 处，其他位置的散度确实为0

> [!PDF|note] [01 Vector Analysis, p.44](Physics/Electrodynamics/chapters/01%20Vector%20Analysis.pdf#page=44&selection=142,30,175,5&color=note)
> >  Notice that the **surface integral (Eq. 1.85) is independent of R**; if the divergence theorem is right (and it is), we should get ∫ (∇ · v) dτ = 4π for any sphere centered at the origin, no matter how small
> 
> 正常情况下散度在一个体积内的积分等于这个区域表面积上的通量

> [!PDF|note] [01 Vector Analysis, p.44](Physics/Electrodynamics/chapters/01%20Vector%20Analysis.pdf#page=44&selection=175,7,197,2&color=note)
> > Evidently, the **entire contribution must be coming from the point r = 0**! Thus, ∇ · v has the bizarre property that **it vanishes everywhere except at one point**, and yet **its integral (over any volume containing that point) is 4π**.

# Summary

[nodes]
- Divergence of $\mathbf{\dfrac{\hat{r}}{r^{2}}}$ 
	- The intergral of the divergence of $\mathbf{\hat{r} / r^{2}}$ over any volume contains the origin point is $\mathbf{4\pi}$ 
- Dirac Delta Function
	- one-dimensional
		- Definition - not a general function
			- $\delta(x) = \left\{\begin{array}{l}0, x \neq 0 \\ \infty, x = 0\end{array}\right.$
		- Properties
			- The **Integral** of a dirac function from $-\infty$ to $\infty$ is **1**
				- $\int_{-\infty}^{\infty}\delta(x)dx = 1$
			- a dirac function can **pick out** the specific value of another function
				- $f(x)\delta(x) = f(0)\delta(x)$ 
			- a dirac function can **shift it peak** and then also pick out the specific value of another function
				- $f(x)\delta(x - a) = f(a)\delta(x - a)$
			- The equivalent condition
				- $\int_{-\infty}^{\infty} f(x)D_{1}(x)dx = \int_{-\infty}^{\infty} f(x)D_{2}(x)dx$
	- three-dimansional
		- Definition
			- $\delta^{3}(\mathbf{r}) = \delta(x)\delta(y)\delta(y)$
		- Properties
			- Integral
				- $\int_{all space}\delta(\mathbf{r})d\tau = \int_{-\infty}^{\infty} \delta(x)dx \int_{-\infty}^{\infty} \delta(y)dy \int_{-\infty}^{\infty} \delta(z) dz = 1$
			- defining conditions
				- $\nabla \cdot (\dfrac{\mathbf{\hat{r}}}{r^{2}}) = 4\pi \delta^{3}(\mathbf{r})$ 
				- $\nabla(\dfrac{1}{r}) = -\dfrac{\mathbf{\hat{r}}}{r^{2}}$ 
				- $\nabla^{2} \dfrac{1}{r} = -4\pi \delta^{3}(\mathbf{r})$

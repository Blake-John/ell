> [!PDF|example] [01 Vector Analysis, p.12](Physics/Electrodynamics/chapters/01%20Vector%20Analysis.pdf#page=12&selection=137,0,174,0&color=example)
> > Suppose we have a function of one variable: f (x). Question: What does the derivative, df /dx, do for us? Answer: It tells us how rapidly the function f (x) varies when we change the argument x by a tiny amount, dx
> 
> ![01 Vector Analysis, p.12](Physics/Electrodynamics/chapters/01%20Vector%20Analysis.pdf#page=12&rect=260,170,326,195&color=example)

> [!PDF|note] [01 Vector Analysis, p.12](Physics/Electrodynamics/chapters/01%20Vector%20Analysis.pdf#page=12&selection=191,0,207,47&color=note)
> In words, if we increment x by an infinitesimal amount dx, then f changes by an amount df ; the derivative is the proportionality factor.

> [!PDF|important] [01 Vector Analysis, p.13](Physics/Electrodynamics/chapters/01%20Vector%20Analysis.pdf#page=13&selection=36,0,57,1&color=important)

# 1.2.2 Gradient

> [!PDF|example] [01 Vector Analysis, p.13](Physics/Electrodynamics/chapters/01%20Vector%20Analysis.pdf#page=13&selection=111,0,119,45&color=example)
> A derivative is supposed to tell us how fast the function varies if we move a little distance. But this time the situation is more complicated, because it depends on what direction we move: if we go straight up, then the temperature will probably increase fairly rapidly, but if we move horizontally, it may not change much at all. 

> [!PDF|note] [01 Vector Analysis, p.13](Physics/Electrodynamics/chapters/01%20Vector%20Analysis.pdf#page=13&selection=126,52,127,23&color=note)
> > A theorem on partial derivatives states that
>

> [!PDF|note] [01 Vector Analysis, p.13](Physics/Electrodynamics/chapters/01%20Vector%20Analysis.pdf#page=13&selection=175,0,189,1&color=note)
> This tells us how T changes when we alter all three variables by the infinitesimal amounts dx, dy, dz.

> [!PDF|note] [01 Vector Analysis, p.13](Physics/Electrodynamics/chapters/01%20Vector%20Analysis.pdf#page=13&selection=198,14,203,34&color=note)
> the partial derivatives along each of the three coordinate directions

> Equation 1.34 is reminiscent of a dot product: dT = ( ∂T ∂x ˆx + ∂T ∂y ˆy + ∂T ∂z ˆz) · (dx ˆx + dy ˆy + dz ˆz) = (∇T ) · (dl),

> [!PDF|def] [01 Vector Analysis, p.14](Physics/Electrodynamics/chapters/01%20Vector%20Analysis.pdf#page=14&selection=4,0,41,0&color=def)
> > where ∇T ≡ ∂T ∂x ˆx + ∂T ∂y ˆy + ∂T ∂z ˆz
> 
> > [!PDF|def] [01 Vector Analysis, p.14](Physics/Electrodynamics/chapters/01%20Vector%20Analysis.pdf#page=14&selection=43,0,50,2&color=def)
> > is called the gradient of T. 

> [!PDF|note] [01 Vector Analysis, p.14](Physics/Electrodynamics/chapters/01%20Vector%20Analysis.pdf#page=14&selection=52,0,60,69&color=note)
> ∇T is a vector quantity, with three components; it is the generalized derivative we have been looking for

> [!PDF|note] [01 Vector Analysis, p.14](Physics/Electrodynamics/chapters/01%20Vector%20Analysis.pdf#page=14&selection=63,0,73,2&color=note)
> Geometrical Interpretation of the Gradient: Like any vector, the gradient has magnitude and direction. 

> [!PDF|example] [01 Vector Analysis, p.14](Physics/Electrodynamics/chapters/01%20Vector%20Analysis.pdf#page=14&selection=73,3,103,0&color=example)
> > o determine its geometrical meaning, let’s rewrite the dot product (Eq. 1.35) using Eq. 1.1: dT = ∇T · dl = |∇T | |dl| cos θ,
> 
> > [!PDF|example] [01 Vector Analysis, p.14](Physics/Electrodynamics/chapters/01%20Vector%20Analysis.pdf#page=14&selection=105,0,118,1&color=example)
> where θ is the angle between ∇T and dl.

> [!PDF|example] [01 Vector Analysis, p.14](Physics/Electrodynamics/chapters/01%20Vector%20Analysis.pdf#page=14&selection=118,7,179,7&color=example)
> > if we fix the magnitude |dl| and search around in various directions (that is, vary θ), the maximum change in T evidentally occurs when θ = 0 (for then cos θ = 1). That is, for a fixed distance |dl|, dT is greatest when I move in the same direction as ∇T. Thus,

> [!PDF|note] [01 Vector Analysis, p.14](Physics/Electrodynamics/chapters/01%20Vector%20Analysis.pdf#page=14&selection=181,0,185,10&color=note)
> the gradient ∇T points in **the direction of maximum increase of the function** T

> [!PDF|note] [01 Vector Analysis, p.14](Physics/Electrodynamics/chapters/01%20Vector%20Analysis.pdf#page=14&selection=189,0,198,17&color=note)
> the magnitude |∇T | **gives the slope (rate of increase) along this maximal direction**

> [!PDF|example] [01 Vector Analysis, p.14](Physics/Electrodynamics/chapters/01%20Vector%20Analysis.pdf#page=14&selection=200,0,215,13&color=example)
> Imagine you are standing on a hillside. Look all around you, and find **the direction of steepest ascent**. That **is the direction of the gradient**. Now measure **the slope in that direction** (rise over run). That **is the magnitude of the gradient**.

> [!PDF|note] [01 Vector Analysis, p.14](Physics/Electrodynamics/chapters/01%20Vector%20Analysis.pdf#page=14&selection=245,0,302,72&color=note)
> What would it mean for **the gradient to vanish**? If ∇T = 0 at (x, y, z), then dT = 0 for small displacements about the point (x, y, z). This is, then, **a stationary point of the function** T (x, y, z). It **could be a maximum (a summit), a minimum (a valley), a saddle point (a pass), or a “shoulder.”**

## properties

> [!PDF|note] [01 Vector Analysis, p.16](Physics/Electrodynamics/chapters/01%20Vector%20Analysis.pdf#page=16&selection=288,11,293,47&color=note)
> > del is **not a vector**, in the usual sense. Indeed, it **doesn’t mean much until we give it a function to act on**

> [!PDF|note] [01 Vector Analysis, p.16](Physics/Electrodynamics/chapters/01%20Vector%20Analysis.pdf#page=16&selection=293,62,301,12&color=note)
> > it **does not “multiply”** T; rather, it is an **instruction to differentiate** what follows

> [!PDF|important] [01 Vector Analysis, p.16](Physics/Electrodynamics/chapters/01%20Vector%20Analysis.pdf#page=16&selection=301,14,317,0&color=important)
> > To be precise, then, we say that ∇ is a **vector operator** that acts upon T, **not a vector** that multiplies T

## Calculation

> [!PDF|note] [01 Vector Analysis, p.16](Physics/Electrodynamics/chapters/01%20Vector%20Analysis.pdf#page=16&selection=376,16,421,2&color=note)
> >  there are three ways the operator ∇ can act: 
> >  1. on a scalar function T: ∇T (the gradient); 
> >  2. on a vector function v via the dot product: ∇ · v (the divergence); 
> >  3. on a vector function v via the cross product: ∇ × v (the curl).
